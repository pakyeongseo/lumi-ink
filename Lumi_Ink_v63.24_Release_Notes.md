# v63.24 — 독립 프레임 정리·정사각형 색상판

## 프레임 정리
- 이미지·영상 조각의 프로젝트 연동 프레임 기능을 제거했습니다.
- 프레임 종류와 색상은 이제 조각별 독립 설정만 저장합니다.
- 기존 `frameLinked` / `useProjectFrame` 데이터는 앱 시작 시 정리되며, 연동 전용으로 저장된 조각은 프레임 없음으로 초기화됩니다.

## 색상 선택기
- 자유 메모와 아이디어 보드의 직접 색상 선택을 정사각형 채도·명도 색상판 + 색조 슬라이더 UI로 교체했습니다.
- HEX, RGB, 기본 팔레트, 저장한 색상은 즉시 동기화됩니다.
- RGB 입력은 R/G/B 테마 그라데이션 태그, 중앙 정렬 값, 테마 기조의 짙은 글자색으로 다듬었습니다.
- 프로젝트 썸네일 프레임과 아이디어 보드 이미지·영상 프레임의 팔레트 마지막에 직접 색상 선택을 추가했습니다.

## 호환성
- 일반 프레임 설정은 그대로 유지됩니다.
- 과거 프로젝트 연동 전용 설정만 정리 대상입니다.
