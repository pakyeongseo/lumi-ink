# 개발 빌드 v66.19 · 코드 작업실 통합 · Markdown 원문 지원

- 기존 사용자 표기 **HTML 작업실**을 **코드 작업실**로 변경했습니다. 내부 메모 타입과 기존 백업 데이터는 호환을 위해 그대로 유지합니다.
- 코드 작업실은 HTML·JSON·Markdown 원문을 동일한 원문 편집기에서 보관·수정·복사·내보낼 수 있습니다.
- 사이드바 `열기`에서 `.md`, `.markdown`, `.mdown`, `.mkdn`, `.mkd`를 인식해 Markdown 원문을 코드 작업실로 바로 엽니다.
- 파일 저장 창에 **HTML `.html` / JSON `.json` / Markdown `.md`** 선택지를 추가했습니다. JSON 저장만 문법 검사를 합니다.
- HTML은 기존 샌드박스 렌더링과 웹페이지 새 창 열기를 유지하며, JSON과 Markdown은 실행하지 않는 안전한 원문 확인 보기로 표시합니다.
- 매뉴얼·README·사용자 릴리스 노트·오프라인 캐시를 새 명칭과 3개 파일 형식 기준으로 갱신했습니다.
