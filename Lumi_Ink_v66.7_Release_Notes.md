# Lumi Ink v66.7 · 사용자 지정 테마 자동 역할 정리

- 사용자 지정 컬러 테마에 다음 색상 역할을 추가했습니다.
  - 자유 메모 코드 보기 아이콘 배경
  - 메인 화면 섹션 제목 배경
  - 설정 카드 그림자
  - 설정 카드 눌림 배경
  - 새 메모 팝업 아이콘 배경
- 퀵메뉴의 기본 기능 아이콘 타일 그라데이션은 별도 설정을 만들지 않고, 사용자 지정 메인 색상 1·2에서 자동 파생합니다.
- 프로젝트 상세의 메모 타입 구분자는 자유 메모·HTML·로어북·로그·페르소나·캐릭터·아이디어 보드 7종을 메인 그라데이션에서 자동 매칭합니다.
  - 타입 태그도 같은 자동 팔레트를 사용합니다.
  - 이전의 공통 구분자 4색 수동 입력은 자동 팔레트 체계로 정리했습니다.
- 사용자 지정 테마 데이터 형식을 v10으로 올렸고, 기존 v9 이하 테마는 새 역할 색상을 추천값으로 자동 보완합니다.

## 검증

- `node --check app.js`
- 사용자 지정 테마 v9 → v10 정규화 및 JSON 보관 구조 확인
- 7개 메모 타입 구분자 데이터 속성·CSS 변수 연결 확인
- 서비스 워커 캐시 버전 및 자산 경로 확인
- ZIP 무결성 검사
