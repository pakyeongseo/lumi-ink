# Lumi Ink v63.37 Release Notes

## 핵심 변경

- 아이디어 보드 이미지 조각의 최소 크기 제한을 완화했습니다.
  - 원본 이미지보다 작게 축소할 수 있습니다.
- 스냅 기능을 리사이즈에도 확장했습니다.
  - 조각을 확대/축소할 때도 주변 조각의 가장자리·중앙 기준에 맞춰 가이드가 표시됩니다.
- 조각별 잠금 기능을 추가했습니다.
  - 잠긴 조각은 이동, 크기 조절, 회전, 삭제, 복제, 순서 변경, 메모지 본문 편집이 막힙니다.
  - 조각 옵션에서 잠금/잠금 해제를 전환합니다.
- 다중선택 모드를 추가했습니다.
  - 상단 다중선택 버튼으로 진입합니다.
  - 빈 캔버스에서 드래그 박스로 선택하거나 조각을 터치/클릭해 선택을 토글합니다.
  - 다중선택 도구에서 전체선택, 좌/중/우 정렬, 상/중/하 정렬을 사용할 수 있습니다.
- 그룹화/그룹 풀기를 추가했습니다.
  - 다중선택한 조각을 그룹으로 묶을 수 있습니다.
  - 그룹 안의 아무 조각을 선택하면 그룹 박스가 표시됩니다.
  - 그룹 박스에서 그룹 단위 이동과 확대/축소가 가능합니다.
  - 그룹 옵션에서 그룹 잠금/해제와 그룹 풀기를 처리합니다.
- 새 조각 타입 `빈 프레임`을 추가했습니다.
  - 장식용 빈 프레임을 보드에 배치할 수 있습니다.
  - 다른 조각처럼 프레임 씌우기와 표시 제목 변경을 지원합니다.
  - 좌우반전은 제공하지 않습니다.
- 메모지 본문 세로 중앙맞춤 옵션을 추가했습니다.
- 메모지 미니 리치툴에 글자색 선택기를 추가했습니다.
  - 선택 영역이 있으면 해당 부분만 바뀝니다.
  - 선택 영역이 없으면 해당 메모지 본문 전체에 적용됩니다.
