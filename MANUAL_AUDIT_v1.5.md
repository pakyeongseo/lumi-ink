# Lumi Ink v1.5 매뉴얼 정합성 점검

- 사용자 배포 표기: **v1.5**
- 개발 기준 표기: **v64.4**
- 점검일: 2026-06-26

## 발견 및 정리한 항목

1. 상세 매뉴얼의 문서 제목·사이드바·헤더가 v63.x/v1.0으로 섞여 있던 문제를 v1.5와 개발 기준 v64.4로 통일했습니다.
2. 새 메모의 실제 3탭, 자유 메모 빠른 생성, 최신 인물 카드 명칭을 본문과 목차에 반영했습니다.
3. 자동 백업의 실제 동선(스냅샷 목록 우선 → 보관 설정 분리)과 1~15개 보관 규칙을 반영했습니다.
4. 아이디어 보드의 다중선택 시각 표시, 그룹 스냅, 잠긴 메모의 상위 레이어 도구, PNG 전용 장면 저장, 스포이드 안내를 추가했습니다.
5. 현재 프로젝트 섹션 순서와 프로젝트 고정 최대 4개를 수정했습니다.
6. 문서 닫는 태그 뒤에 붙어 있던 과거 변경 이력 조각을 제거하고, 현재 기준을 문서 본문 안의 별도 섹션으로 넣었습니다.
7. README와 최신 사용자 릴리스 노트를 v1.5 기준으로 새로 정리했습니다.

## 범위

이번 패키지는 기능 로직을 변경하지 않았습니다. 매뉴얼·README·릴리스 노트·서비스워커 캐시 메타데이터를 현재 구현에 맞춰 정리한 문서 정합성 패키지입니다.
