# Lumi Ink v65 — 사용자 지정 컬러 테마

> 이 구현의 전역 자동 팔레트 방식은 v65.1에서 구조 색상 개별 지정 방식으로 교체되었습니다. 최신 변경은 `Lumi_Ink_v65.1_Release_Notes.md`를 참고하세요.

사용자 배포 버전은 **v1.5**이며, 이 문서는 개발 추적용 **v65** 변경 기록입니다.

## 사용자 지정 컬러 테마

- 설정 → **컬러 테마**에 **직접 지정** 항목을 추가했습니다.
- 주조색과 보조색을 각각 색상 선택기 또는 `#RRGGBB` HEX 값으로 입력할 수 있습니다.
- 두 색을 바꾸는 동안 앱 전체에 실시간 미리보기가 적용됩니다.
  - 취소하면 적용 전 프리셋 또는 기존 직접 지정 테마로 안전하게 되돌아갑니다.
  - 보조색은 **보조색 자동** 버튼으로 주조색의 색상 관계에 맞춰 다시 생성할 수 있습니다.
- 밝게/어둡게 모드마다 같은 색을 그대로 쓰지 않고, 읽기성과 표면 대비를 유지하도록 강조색·보조색·배경·테두리·글자 색을 자동 조율합니다.
- 로고, 기본 썸네일, 버튼 그라데이션, 퀵 메뉴, 로어북·페르소나 헤딩 등 기존 테마 토큰을 사용하는 화면이 커스텀 팔레트를 함께 따릅니다.

## 저장·백업

- 직접 지정 색상은 IndexedDB 설정 저장소와 로컬 설정에 함께 저장됩니다.
- 전체 백업과 자동 백업 payload를 v3으로 확장해 현재 선택한 컬러 테마와 직접 지정 색상도 함께 보관합니다.
- v65 이전 백업은 기존처럼 그대로 복원됩니다. 색상 정보가 없는 과거 백업을 복원할 때는 현재 앱의 컬러 테마를 유지합니다.

## 배포 갱신

- 서비스 워커 캐시 키를 `ink-memo-v1.5-v65-custom-theme`으로 갱신했습니다.
- 사용자 배포 표기는 계속 **v1.5**이며, 내부 개발 기준만 **v65**로 올렸습니다.
