# Lumi Ink 개발 빌드 v64.4 — 자동 백업 목록 우선 · 새 메모 팝업 투명 칩

> 이 문서는 개발 변경 이력입니다. 사용자 배포 표기는 **v1.5**입니다.

## 변경 사항

### 자동 백업
- 설정의 **자동 백업**을 누르면 보관 개수 편집창 대신 **스냅샷 목록**을 먼저 엽니다.
- 스냅샷이 아직 없어도 동일한 목록 화면을 표시하며, 하단의 **보관 설정**에서만 1~15개 보관 수를 조절할 수 있습니다.
- 보관 설정 창에 **목록으로** 동선을 추가해 설정을 확인한 뒤 스냅샷 목록으로 돌아갈 수 있습니다.

### 새 메모
- 상단 안내 문구를 `생성할 메모 타입을 골라주세요`로 변경했습니다.
- 탭 안의 메모 타입 카드는 카드 면과 아이콘 면을 투명 처리했습니다. 팝업 표면이 그대로 드러나며, 다크 테마에서는 제목·설명·화살표가 밝은 잉크색으로 읽히도록 보정했습니다.
- 상단 **자유 메모** 빠른 생성 카드는 라이트 테마의 밝고 가벼운 기조를 라이트/다크 모드 공통으로 유지합니다. 선택한 색상 테마의 기조만 은은하게 반영합니다.

## 배포
- 서비스워커 캐시 키를 `v64.4-backup-list-transparent-chips`로 갱신했습니다.
