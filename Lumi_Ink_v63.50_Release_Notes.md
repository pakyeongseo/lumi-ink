# Lumi Ink v63.50 변경사항

## 아이디어 보드 보수

- 보기 모드의 현재 화면 PNG 저장에서 일부 조각이 깨져 보이던 문제를 보정했습니다.
- 화면 스포이드 fallback 캡처도 PNG 저장과 같은 안전 캡처 경로를 사용하도록 수정했습니다.
- 캡처 복제본에서 `color-mix()`·`color()`가 포함된 일반 요소와 `::before`/`::after` 장식 요소의 색상을 안정적으로 변환합니다.
- 구분선 옵션 메뉴에서 별도 컬러 버튼을 제거했습니다.
- 구분선은 디자인을 고른 뒤 바로 컬러 선택 팝업으로 이어지며, 컬러 단계에는 뒤로가기 버튼을 추가했습니다.
- 다른 메모를 열었다가 브라우저 뒤로가기로 아이디어 보드에 돌아왔을 때 보드가 동작하지 않던 라우팅 상태 복원 문제를 수정했습니다.
- 서비스워커 캐시명을 갱신해 배포 후 이전 스크립트가 남지 않도록 했습니다.
