# 개발 빌드 v66.25 · 정규식 작업실 미리보기 보정

- 정규식 작업실 미리보기의 `$1`, `$2`, `$3` 캡처 대응 목록을 작은 매핑 칩 형태로 정리했습니다. 캡처가 없을 때는 빈 섹션이 남지 않습니다.
- 미리보기 상태 표기를 큰 배지 디자인에서 섹션 제목 옆의 작은 상태 텍스트로 되돌렸습니다.
- `샘플→IN` 실행 후에도 sample 입력칸의 `[[바뀌는 값]]` 표기가 유지되도록 수정했습니다. 미리보기에서는 기존처럼 `[[ ]]` 표식을 제거한 값만 사용합니다.
- 서비스 워커 캐시명을 갱신해 모바일/PWA 환경에서 이번 수정본이 이전 캐시에 묶이지 않도록 했습니다.
