# Lumi Ink v63.38 Release Notes

## 핵심 변경

- 아이디어 보드 `내 메모 링크` 선택 팝업 디자인을 개편했습니다.
  - 로그 저장 메모 타입의 템플릿 선택기와 같은 탭형 레이아웃을 사용합니다.
  - `전체보기` 탭에서 연결 가능한 모든 메모를 바로 선택할 수 있습니다.
  - `프로젝트별` 탭에서 현재 존재하는 프로젝트를 먼저 고른 뒤, 해당 프로젝트의 메모를 선택할 수 있습니다.
  - 기존 제목·내용·종류 검색 기능은 유지했습니다.
- 아이디어 보드 보기 전용 모드 진입 방식을 변경했습니다.
  - 최상단 바의 눈 아이콘으로 보기 전용 모드에 들어갑니다.
  - 보기 전용 모드에서는 같은 버튼이 연필 아이콘으로 바뀌며, 누르면 편집 모드로 돌아옵니다.
  - 보드/아티팩트 탭은 편집 화면 안의 표시 방식 전환으로 유지했습니다.
- 보드별 스냅 선택 상태 저장을 추가했습니다.
  - 스냅 켜짐/꺼짐 상태가 각 아이디어 보드의 `canvas.snapOn`에 저장됩니다.
  - 다른 보드로 이동했다가 돌아와도 해당 보드의 스냅 상태가 복원됩니다.
