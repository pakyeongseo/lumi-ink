# Lumi Ink v66.39 릴리스 노트

## 퀵메뉴 아이콘 — 배지 배경 가독성 보정

- 퀵메뉴와 프로젝트 썸네일에 등록한 아이콘의 **배지 배경**이 본체와 같은 진하기로 칠해져, 컬러를 적용했을 때 아이콘 본체 모습이 배경에 묻히던 문제를 고쳤습니다.
- 배지 배경 그라데이션에만 투명도(`fill-opacity`)를 더해 살짝 옅고 비치게 만들었습니다. 배경이 표면색과 자연스럽게 섞이면서, 불투명한 아이콘 본체가 위로 또렷하게 떠 보입니다.
- 색상 변수 자체(base·deep 등)는 건드리지 않았기 때문에, 종이 그림자나 입체 디테일 같은 다른 표현은 그대로 유지됩니다.

## 퀵메뉴 아이콘 — 등록 시 생기던 라인 제거

- 아이콘을 슬롯에 등록하면 가장자리에 보이던 **정체불명의 라인**의 원인을 추적해 제거했습니다.
- 원인은 각 아이콘 SVG가 가진 **외곽 광택선**이었습니다. 슬롯 전용으로 적용되는 미세 확대(scale) 때문에 이 광택선이 배지 가장자리에 바짝 붙으면서 프레임처럼 읽혔습니다.
- 외곽 광택선만 정확히 골라 숨겼습니다. 돋보기 렌즈 하이라이트 같은 **내부 디테일은 그대로 보존**됩니다.
- 아이콘 라이브러리의 선택 그리드(더 큰 미리보기)에는 이 처리를 적용하지 않아, 라이브러리에서는 기존과 동일한 광택선을 그대로 볼 수 있습니다.

## 기술 메모

- 적용 범위: 퀵메뉴 슬롯, 퀵메뉴 매니저 슬롯, 퀵메뉴 항목 미리보기, 선택지·함수 행, 프로젝트 썸네일.
- 기존 아이콘 라이브러리(70종) SVG 데이터와 슬롯 등록 데이터 구조는 변경하지 않았습니다.
- 서비스 워커 캐시 키를 `v66.39-quickmenu-icon-badge-line`으로 갱신했습니다.
