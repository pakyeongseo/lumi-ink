# Lumi Ink v63.29 변경사항

## UI 미감 다듬기

- **형광펜 버튼 재설계 (에디터 · 크리에이터 메모)** — 하단을 노랗게 채우던 방식이 둥근 모서리에 잘려 '바구니'처럼 보이던 문제를 고쳤습니다. 이제 다른 도구와 같은 라인 아이콘(마커)에 현재 형광펜 색을 보여주는 잉크 밑줄을 더한 형태로 표시됩니다. 색상 선택 기능은 그대로입니다.
- **아이디어 보드 보기 전환** — 좁은 화면에서 `아티팩트` 탭 글자가 잘리던 문제를 고쳤습니다. 보드/아티팩트 토글은 왼쪽에 고정되고, 줌·되돌리기 등 도구는 가로로 스크롤됩니다.
- **아이디어 보드 줌** — `100%로 초기화` 버튼 라벨을 `100`에서 `1:1`로 바꿔 현재 배율 표시(`100%`)와 헷갈리지 않도록 했습니다.
- **HTML 작업실 RAW 안내** — 좁은 화면에서 설명 문구가 어중간하게 잘려 보이던 부분을 정리했습니다. 모바일에서는 `RAW` 배지만 표시하고, 전체 설명은 길게 눌렀을 때(툴팁)로 확인할 수 있습니다. 문구도 '정화하지 않고' → '필터링 없이'로 다듬었습니다.
- **브랜드 로고 별 통일** — 헤더 로고의 별을 푸터 및 브랜드 표기(`Lumi ✦ Ink`)와 동일한 4각 스파클로 맞췄습니다.

## 참고

- 동작·데이터에는 변화가 없으며 표시(스타일)만 다듬은 업데이트입니다.
