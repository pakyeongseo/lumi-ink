# Lumi Ink v63.43 — 메모지 잠금·4방향 핸들·스포이드

## 반영
- 메모지 잠금을 `요소만 잠금`과 `전체 보호 잠금`으로 분리했습니다.
  - 요소만 잠금: 이동, 크기, 회전, 삭제, 디자인 변경을 보호하고 메모 본문 편집은 유지합니다.
  - 전체 보호 잠금: 본문을 포함한 모든 편집을 보호합니다.
- 선택 핸들의 상단 순서를 `이동 → 메뉴`로 정리하고, 네 모서리에 짧고 두꺼운 리사이즈 그립을 배치했습니다.
- 모든 정사각형 직접 색상 선택기에 화면 색상 스포이드를 추가했습니다. (지원 브라우저/HTTPS에서 사용 가능)
- 메모 링크 프로젝트 목록의 썸네일을 고정 정사각형·cover 렌더링으로 보정했습니다.
- 메모지 본문은 스크롤을 유지하되 스크롤바를 감췄습니다.
- 아이디어 보드·HTML 작업실 구분자는 현재 테마의 색상 기조에서 서로 다른 톤을 사용하도록 맞췄습니다.

## 검증 범위
- JavaScript 문법 검사
- 서비스 워커 캐시 버전 및 자산 경로
- ZIP 무결성, 중복·위험 경로 검사

## 참고
- 화면 색상 추출은 브라우저의 `EyeDropper` API를 사용합니다. 기기·브라우저 또는 비보안 HTTP 환경에서는 아이콘이 비활성화되고, 기존 정사각형 색상판·HEX·RGB 입력은 그대로 사용할 수 있습니다.
