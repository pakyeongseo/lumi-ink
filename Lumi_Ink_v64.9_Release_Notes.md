# Lumi Ink v64.9 — 퀵 메뉴 표시 설정

사용자 배포 버전은 **v1.5**이며, 이 문서는 개발 추적용 **v64.9** 변경 기록입니다.

## 퀵 메뉴 사용 및 표시 형태

- 설정 → **퀵 메뉴 설정**에서 퀵 메뉴의 사용 여부를 켜거나 끌 수 있습니다.
  - 끄면 오른쪽 엣지 탭과 슬라이드 제스처가 숨겨집니다.
  - 슬롯 구성, 사용자 지정 썸네일, 등록 대상은 삭제되지 않고 그대로 보관됩니다.
- 표시 형태를 두 가지 중에서 선택할 수 있습니다.
  - **기본형**: 아이콘, 제목, 설명, 슬롯 번호를 함께 표시합니다. 기본값입니다.
  - **미니형**: 설명·제목 없이 아이콘 전용 슬롯만 보이는 압축형입니다. 슬롯 편집용 설정 버튼은 유지됩니다.
- 설정값은 기존 퀵 메뉴 데이터와 함께 IndexedDB, 전체 백업, 자동 백업에 저장됩니다. 기존 v64.8 구성은 자동으로 `켜짐 + 기본형`으로 승격됩니다.

## 디자인 조정

- 라이트 테마에서 퀵 메뉴의 제목 글자를 순수 검정 대신, 선택한 컬러 테마를 눌러 만든 짙은 잉크색으로 표시합니다.
- 퀵 메뉴와 탭의 외곽 그림자 블러 범위와 불투명도를 대폭 낮췄습니다.
- 펼친 메뉴 안의 편집 버튼을 홈 화면 설정 버튼과 같은 기어 실루엣으로 교체하고, 작은 그라데이션 버튼 표면으로 정리했습니다.
