# Lumi Ink v63.46 — Idea Board Rendering Fixes

기준 소스: v63.45 아이디어 보드 수정본

## 수정 사항

- **다중 선택 표시 즉시 반영**
  - 선택 직후 각 조각을 상위 레이어로 올리고, 외곽선·광륜을 즉시 표시합니다.
  - 잠금·그룹 조각도 동일하게 선택 상태가 눈에 보입니다.

- **색상 스포이드 안정화**
  - 중복 클릭 바인딩을 제거해 `EyeDropper.open()`이 두 번 호출되며 취소되는 문제를 정리했습니다.
  - 지원 브라우저에서는 화면 스포이드를 사용하고, 미지원 환경/PWA에서는 기본 색상 선택기로 자연스럽게 전환합니다.

- **내 메모 링크 → 프로젝트 탭 썸네일 프레임 수정**
  - 프로젝트 카드·사이드바와 같은 프레임 렌더러(`frameInner`)를 사용하도록 통일했습니다.
  - 미디어는 내부에서 클리핑하고 프레임은 바깥 레이어에서 렌더링해 프레임 비율과 모서리 잘림을 바로잡았습니다.

- **잠긴 메모지 서식 툴바 레이어 보정**
  - 잠금된 메모가 선택되거나 포커스를 얻는 순간, 메모 본체와 리치 서식 툴바를 다른 조각보다 위 레이어에 배치합니다.

- **현재 장면 PNG 저장 복구**
  - `foreignObject → SVG` 우회 경로 및 SVG 대체 저장을 제거했습니다.
  - 화면에 렌더링된 보드 DOM을 오프스크린 캡처 뷰포트에 복제한 뒤 `html2canvas`로 직접 PNG 래스터화합니다.
  - 실패 시에도 SVG를 저장하지 않고 명확한 PNG 오류 안내만 표시합니다.

- **그룹 이동·크기 조절 스냅 적용**
  - 그룹의 외곽 경계 전체가 캔버스 중심선·다른 조각의 좌/중앙/우 및 상/중앙/하 기준선에 맞춰집니다.
  - 그룹 내부 멤버는 스냅 대상에서 제외하여 자기 자신에게 달라붙는 현상을 막았습니다.
