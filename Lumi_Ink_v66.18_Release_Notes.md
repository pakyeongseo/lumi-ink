# 개발 빌드 v66.18 · 사용자 테마 공통 옅은 강조 배경

- 사용자 지정 컬러 테마에 **옅은 강조·선택 배경** 단일 색상 역할을 추가했습니다.
- 이 역할은 런타임의 `--accent-soft`를 직접 설정하며, 코드 보기 아이콘 배경, 설정의 글자 크기 선택, 새 메모 상단 빠른 생성 아이콘·타입 아이콘, 섹션 제목, 메모 개수, 정렬 버튼, 첨부·태그·칩·탭·선택 상태, 아이디어 보드의 공통 강조 배경까지 모두 반영됩니다.
- 사용자 지정 테마에서 별도 변수로 덮어쓰던 작은 섹션 제목·사이드바 메모 개수·코드 보기 아이콘·메인 섹션/정렬·새 메모 아이콘·설정 눌림 배경도 이 공통 역할을 강제로 공유하도록 정리했습니다.
- 기존 테마 JSON에는 기존의 작은 섹션 제목 배경 값을 우선 승계해 새 색상 역할로 마이그레이션합니다. 기존 색상 데이터가 없어도 메인 그라데이션 기반 추천값으로 안전하게 채워집니다.
