# Lumi Ink v63.30 변경사항

## 색상 선택기 정리

글자 색 · 프레임 색 · 메모지 색 등에서 공통으로 쓰는 색상 선택기를 더 깔끔하고 한눈에 들어오도록 다듬었습니다.

- **세로로 길어 `취소·적용` 버튼이 화면 밖으로 밀리던 문제**를 해결했습니다. 이제 한 화면 안에서 색을 고르고 바로 적용할 수 있습니다.
- 레이아웃을 위에서 아래로 흐르는 단순한 구성으로 바꿨습니다: 색상판 → 색조 막대(가로 전체) → 미리보기 → HEX·색상 선택 → RGB → 기본 색상 → 내 색상.
- **색조 막대를 가로 전체로** 넓혀 더 정확하게 고를 수 있게 했습니다.
- **RGB 입력 칸**을 균일하고 차분한 디자인으로 정리했습니다. (이전의 제각각인 컬러 라벨 박스 제거)
- 기본 색상 팔레트를 더 촘촘하게 배치하고, 불필요한 안내 문구를 정리해 전체 높이를 줄였습니다.

## 참고

- 색 선택 · HEX/RGB 입력 · 저장 등 모든 기능은 그대로이며 표시(디자인)만 다듬은 업데이트입니다.
