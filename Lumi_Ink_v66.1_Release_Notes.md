# Lumi Ink v65.10 / v66.1 — 사용자 테마 색상 보관 · 로그 이름 중복 안내

- 사용자 지정 테마에 `작은 섹션 제목 배경`, `메모 제목` 색상 역할을 추가했습니다.
- 라이트/다크 각각의 사용자 테마를 독립 JSON 파일로 저장하고, 다시 불러와 적용할 수 있습니다.
- 자유 색상 편집기의 **내 색상**은 길게 누르거나 마우스 오른쪽 버튼으로 눌러 삭제할 수 있으며, 삭제 전 확인을 거칩니다.
- 퀵메뉴의 내장 SVG·코드 아이콘은 적용 시 자동 테두리·컨테이너 그림자를 덧입히지 않습니다.
- 로그 이름 바꾸기 세트에서 다른 세트와 같은 원본 이름을 입력하면 즉시 안내하고, 저장 전에 수정으로 돌아가거나 의도한 중복으로 저장할 수 있습니다.
