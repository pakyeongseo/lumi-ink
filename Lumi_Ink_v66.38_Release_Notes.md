# Lumi Ink v66.38 릴리스 노트

## 정규식 작업실 — 상태 정보 위계 보정

- PREVIEW 헤더의 설명 문장(예: `매치 · OUT HTML을 샌드박스에서 렌더링했습니다.`)을 기본 안내 글자 크기로 복원했습니다.
- 작게 유지하는 항목은 **IN 카드 제목 오른쪽의 실시간 검증 상태**와 **PREVIEW 설명 왼쪽의 가변 매치 상태 배지**로 한정했습니다.
- `트림아웃`, `영향을 미침`, `최소 깊이`, `그 외 옵션`, `Ephemerality`, `Macros in Find Regex` 등 옵션 소제목 칩의 실제 글자·여백·높이를 함께 더 작게 압축했습니다.

## 자유 메모 — 내 글 바로가기 폭 보정

- 아이디어 보드형 메모 인용 카드 디자인은 유지하되, 자유 메모 본문에서 전체 폭을 차지하지 않도록 최대 폭을 `520px`으로 제한했습니다.
- 좁은 모바일 화면에서는 부모 본문 폭을 넘지 않으며, 넓은 화면에서는 적당한 카드 폭을 유지합니다.

## 기술 메모

- 기존 메모 링크의 ID·색상 데이터와 정규식 JSON 데이터 구조는 변경하지 않았습니다.
- 서비스 워커 캐시 키를 `v66.38-regex-status-link-width`로 갱신했습니다.
