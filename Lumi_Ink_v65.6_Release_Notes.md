# Lumi Ink v65.6 Release Notes

> 사용자 배포 버전: v1.5 · 개발 버전: v65.6

## 독립 사용자 지정 테마
- 라이트/다크 모드별 사용자 지정 테마에 **메인 색상** 역할을 추가했습니다.
- 사용자 지정 테마는 프리셋을 단순 변형하는 방식이 아니라, 선택한 메인 색상에서 출발하는 독립 팔레트로 동작합니다.
- `메인 색상으로 추천 팔레트 만들기` 버튼으로 앱 배경, 카드, 상단바, 글자, 경계선을 한 번에 추천 값으로 채웁니다. 이후 각 역할 색상은 정밀 에디터에서 개별 보정할 수 있습니다.
- 사용자 지정 테마가 활성화된 모드에서는 accent, accent-2, accent-deep, accent-soft, 로고/버튼 그라데이션도 해당 독립 팔레트의 메인 색상에 맞춰 함께 바뀝니다.

## SVG 아이콘 팔레트
- 기존 30종과 테마형 추가 40종을 포함한 70종 아이콘이 밝은색·중간색·딥컬러·강조색·종이색 토큰을 조합하도록 보강했습니다.
- 라이트/다크 모드와 사용자 지정 테마에서도 평면적으로 흐려지지 않도록 대비를 조정했습니다.
- 벡터 아이콘의 하드코딩 색상은 계속 사용하지 않으며, 모든 아이콘은 루미잉크 컬러 변수만 읽습니다.

## 안전성
- 기존 프리셋은 사용자 지정 테마가 비활성화된 상태에서 원래 색상·그라데이션을 그대로 유지합니다.
- 기존 사용자 지정 테마는 메인 색상 역할이 추가된 v4 데이터로 자동 호환됩니다.
