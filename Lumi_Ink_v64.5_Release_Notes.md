# Lumi Ink 개발 빌드 v64.5 — 새 메모 팝업 시각 복원

> 이 문서는 개발 변경 이력입니다. 사용자 배포 표기는 **v1.5**입니다.

## 변경 사항

### 새 메모 팝업
- v64.2의 원래 시각 언어를 복원했습니다. 상단 빠른 생성 카드는 기존 블루 그라데이션과 흰색 아이콘·텍스트를, 타입 카드는 기존 카드 배경과 아이콘 칩의 배경색·아이콘 색상을 다시 사용합니다.
- v64.4에서 추가된 타입 카드·아이콘 칩 투명 처리, 라이트 고정 빠른 생성 카드, 테마색 제목/설명 오버라이드는 제거했습니다.
- 요청한 조정만 유지했습니다.
  - 타입 카드 폭을 100%로 고정
  - 페르소나 설명을 `단일 페르소나 카드 · 국문/영문 · 이미지`로 변경
  - 상단 빠른 생성 카드 제목을 `자유 메모`로 변경
  - 상단 카드 아래 그림자를 짧고 옅게 조정
- 상단 안내 문구는 `생성할 메모 타입을 골라주세요`로 유지했습니다.

### 유지한 기능
- 자동 백업의 **스냅샷 목록 우선 → 보관 설정 별도 진입** 동선은 유지합니다.
- 사용자 배포 버전 표기는 계속 **v1.5**입니다.

## 배포
