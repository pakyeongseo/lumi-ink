# Lumi Ink v66.0 — 로그 이름 바꾸기 세트 · 상단 제목 빠른 수정

개발 빌드: **v66.0**  
사용자 배포 버전: **v1.5**

## 로그 저장 — 이름 바꾸기 세트 5개

- 로그 상단의 기존 인라인 이름 가림 입력 영역을 **이름 바꾸기 세트 5개**로 재구성했습니다.
- 각 세트는 **원본 이름 최대 20개 → 바꿀 이름 1개** 구조입니다.
- 세트 버튼을 누르면 전용 팝업에서 원본 이름 목록과 바꿀 이름을 편집할 수 있습니다.
- 이름 치환은 원본 로그를 바꾸지 않고, 템플릿 미리보기·인라인 HTML·내보낸 로그 화면에만 적용됩니다.
- 각 세트는 기존 로그 템플릿의 이름 강조 스타일을 그대로 공유합니다.
- v65.9 이하의 `personaNames` / `personaAlias` 데이터는 자동으로 1번 세트로 이전됩니다.

## 모든 메모 — 상단 제목 빠른 수정

- 자유 메모, HTML 작업실, 로어북, 로그, 페르소나, 캐릭터, 아이디어 보드의 최상단 제목을 **더블클릭 또는 더블터치**하면 바로 이름 변경 팝업을 엽니다.
- 이 방식으로 설정한 제목은 메뉴의 이름 바꾸기와 동일하게 `titleLocked` 상태로 저장되므로, 본문·이름 필드·로그 원문이 바뀌어도 사용자가 다시 수정하기 전까지 자동 변경되지 않습니다.
