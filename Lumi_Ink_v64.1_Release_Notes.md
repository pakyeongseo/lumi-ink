# Lumi Ink v64.1 변경사항

## 아이디어 보드 — 스크린샷 / 스포이드 렌더링 정정

스크린샷 PNG 저장과 스포이드 색상 추출 화면에서 일부 디자인이 깨지던 문제를 고쳤습니다. 캡처 도구가 그리지 못하던 요소를 캡처 직전에 실제와 동일한 형태로 변환해 넣도록 처리했습니다.

- **괘선·결 텍스처 복원** — 카세트 라벨·리갈패드·장부 등의 줄선(괘선), 클립 메모의 종이결 같은 미세한 반복 선 무늬가 캡처에서 사라지던 문제를 해결했습니다. 해당 무늬를 한 주기 타일 이미지로 구워 그대로 캡처되게 했습니다.
- **그라데이션 음영 깨짐 해결** — 클립 메모 우상단 하이라이트처럼 투명으로 흐려지는 그라데이션이 캡처에서 회색 그림자처럼 어둡게 번지던 문제를 고쳤습니다. 투명 부분을 같은 색의 투명도 0으로 바꿔 색조를 유지한 채 자연스럽게 사라지게 했습니다.
- 위 보정은 스크린샷 저장과 스포이드 색상 추출 화면에 동일하게 적용됩니다.
