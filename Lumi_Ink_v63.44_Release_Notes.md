# Lumi Ink v63.44 Release Notes

기준 빌드: `lumi-ink-v63.43-note-lock-handles-eyedropper`

이번 빌드는 v63.39~v63.43 변경 이력을 기준으로 기능 상태를 검수하고, 누락되어 있던 아이디어 보드 기능과 배포 정리를 통합한 버전입니다.

## 아이디어 보드

- HTML 내보내기에 표시 범위 선택을 추가했습니다.
  - `현재 보이는 화면`: 상단 바를 제외한 현재 작업 영역만 시각적으로 보이게 저장합니다.
  - `전체 화면`: 보드 전체가 보이게 저장합니다.
  - 두 방식 모두 복원용 JSON 데이터와 첨부 파일 데이터는 전체 보드를 그대로 포함합니다.
- 보기 모드 전용 스크린샷 버튼을 추가했습니다.
  - 현재 보이는 작업 영역만 PNG 이미지로 저장합니다.
- 구분선 요소를 추가했습니다.
  - 기본 템플릿: `실선`, `점선`
  - 빈 프레임과 비슷한 장식 요소이며, 색상·크기·잠금·복제·삭제 흐름을 따릅니다.
  - 추가 템플릿 제작용 `idea-divider-template-guide.md`를 포함했습니다.
- 메모지 미니 리치 에디터에서 글자색 적용 시 선택된 범위만 바뀌도록 조정했습니다.
  - 선택 범위가 없으면 전체 본문을 바꾸지 않고 먼저 글자를 선택하라는 안내를 표시합니다.

## v63.39~v63.43 검수 반영

다음 변경사항이 현재 빌드에 들어간 상태임을 확인했습니다.

- 잠긴 조각의 평상시 자물쇠·점선 숨김 및 선택 시 표시
- 다중선택 터치 누적 선택/해제
- 하단 6버튼 툴바
- 전체선택 토글, 정렬·간격 패널
- 스냅 ON/OFF 축약 표시
- 스냅 리사이즈 즉시 반영
- 그룹 리사이즈 핸들 재생성 흔들림 완화
- 그룹 조각 선택 시 그룹 전체 선택
- 메모 링크 프로젝트 썸네일 표시 및 보정
- 프로젝트 내부 선택 화면의 뒤로가기 배치
- 빈 프레임 옵션 정리
- 이미지·음악·동영상·첨부파일·메모 링크 교체
- 홈 프로젝트 고정 한도 4개
- 메모지 잠금: 요소만 잠금 / 전체 보호 잠금 분리
- 핸들 순서 조정 및 네 모서리 리사이즈
- 직접 색상 선택기 EyeDropper 지원
- 긴 메모지 본문 스크롤바 숨김
- 아이디어 보드·HTML 작업실 구분자 테마 톤 적용

## 배포 정리

- 서비스워커 캐시명을 `ink-memo-v63.44-integrated-export-divider`로 갱신했습니다.
- 최신 릴리즈 노트와 구분선 템플릿 가이드를 캐시 대상에 포함했습니다.
- 배포 패키지에서는 과거 v63.xx 매뉴얼 사본, 과거 릴리즈 노트, 임시 패치 스크립트를 제거하는 방향으로 정리했습니다.
