# Lumi Ink v65.4 변경사항

## 컬러 테마
- v64.9 기준의 프리셋 색상 토큰을 기준으로 복원하고, 기본 로고·버튼·아이콘·그라데이션은 사용자 지정 테마가 덮어쓰지 않도록 분리했습니다.
- 사용자 지정 테마를 라이트 모드 / 다크 모드 탭으로 나눴습니다.
- 각 구조 색상은 자유메모 글자색과 동일한 정밀 색상 에디터(색상판, 색조, HEX, RGB, 저장색, 스포이드)를 통해 설정합니다.

## 아이콘
- 내장 SVG 아이콘 30종의 팔레트를 테마 기반 CSS 변수로 전환했습니다.
- 프로젝트 썸네일 선택 탭 이름을 `아이콘`으로 정리했습니다.

## 퀵 메뉴
- 숨김 손잡이는 계속 최소 노출 상태로 유지됩니다. 화면 오른쪽 가장자리에서 왼쪽으로 슬라이드하면 메뉴가 열립니다.
