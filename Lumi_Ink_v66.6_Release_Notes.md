# Lumi Ink v66.6 — 아이콘 라이브러리·7 슬롯·로그 빠른 편집

- 퀵메뉴 벡터 아이콘 70종을 새 v2 원본으로 전면 교체했습니다. 아이콘 내부의 모든 팔레트는 런타임 컬러 변수로 변환되어, 프리셋과 사용자 지정 두 색 그라데이션에 맞춰 함께 바뀝니다.
- 퀵메뉴 슬롯을 최대 7개로 확장했습니다. 작은 화면에서는 슬롯 목록만 내부 스크롤되어 오른쪽 메뉴의 전체 높이가 화면을 벗어나지 않습니다.
- 로그 저장의 보기 모드에서 본문을 더블클릭 또는 더블터치하면 원본 편집 모드로 즉시 전환됩니다.
- 사용자 지정 컬러 테마에 프로젝트 상세의 메모 타입 구분자 배경·글자·점·개수 색상을 추가했습니다. 라이트/다크 추천 팔레트와 테마 JSON에도 함께 저장됩니다.
