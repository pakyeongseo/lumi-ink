# Lumi Ink v63.41 — Note Menu Cleanup & Board Selection Stability

## 핵심 변경

- 메모 안의 보기/편집 더보기 메뉴에서 `선택` 항목 제거
  - 홈·프로젝트 화면의 다중선택 관리 기능은 유지
- 아이디어 보드·HTML 작업실의 메모 구분 마크에 테마 기조를 공유한 비중첩 색상 적용
  - 아이디어 보드: 골드 계열
  - HTML 작업실: 시안 계열
- 아이디어 보드 메모지 리치 도구 글자 크기를 `A-- / A- / A / A+ / A++ / A+++` 6단계로 세분화
- 메모지 글자색 길게 누르기에서 자유메모 리치 에디터와 동일한 정사각형 색상판·색조 슬라이더·HEX·RGB 입력기 사용
- 다중선택 조각 터치를 캡처 단계에서 우선 처리
  - 내부 이미지·음악·영상·링크·첨부 버튼 위 터치도 선택/해제로 동작
  - 그룹은 기존처럼 단위 전체가 함께 선택
- 아이디어 보드 보기 모드에서 다중선택 아이콘 숨김

## 호환성

- 기존 아이디어 보드 데이터와 조각 위치·프레임·잠금·그룹 데이터는 변경하지 않습니다.
- 이 버전은 UI 이벤트 우선순위와 메뉴 노출만 정리합니다.
