# Lumi Ink v64.8 — 오른쪽 퀵 메뉴

사용자 배포 버전은 **v1.5**이며, 이 문서는 개발 추적용 **v64.8** 변경 기록입니다.

## 신기능 · 오른쪽 엣지 퀵 메뉴

- 모든 화면의 오른쪽 가장자리에 작게 숨겨지는 퀵 메뉴 탭을 추가했습니다.
- 탭을 누르거나 오른쪽 끝에서 왼쪽으로 슬라이드하면 메뉴가 펼쳐집니다. 열린 메뉴는 패널 안에서 오른쪽으로 밀어 닫을 수 있습니다.
- 최대 5개 슬롯을 독립적으로 등록하고, 빈 슬롯은 즉시 설정 화면으로 이어집니다.
- 각 슬롯은 기본 아이콘 또는 사용자 지정 정사각 썸네일을 사용합니다. 업로드 이미지는 320px 정사각 썸네일로 중앙 크롭·압축 저장됩니다.

## 지원하는 바로가기

- 홈 바로가기
- 프로젝트 바로가기
  - 프로젝트가 삭제되면 해당 슬롯을 자동으로 비웁니다.
- 내 글 바로가기
  - 원글이 삭제되면 해당 슬롯을 자동으로 비웁니다.
  - 메모가 다른 프로젝트로 이동해도 메모 ID를 참조하므로 새 경로를 자동으로 따라갑니다.
- 메모 바로 만들기
  - 자유 메모, HTML 작업실, 로어북, 로그 저장, 단일/다인 페르소나, 단일/다인 캐릭터, 아이디어 보드를 지원합니다.
  - 현재 프로젝트 또는 지정 프로젝트에 바로 만들 수 있습니다. 지정 프로젝트가 삭제되면 현재 프로젝트/선택 방식으로 안전하게 되돌립니다.

## 데이터 보존

- 퀵 메뉴 구성은 IndexedDB의 전용 설정 저장소에 보관합니다.
- 전체 백업과 자동 백업에 퀵 메뉴 구성을 함께 포함합니다.
- v64.8부터 DB 스키마 버전이 5로 올라가며, 기존 프로젝트·메모 데이터는 그대로 유지됩니다.
