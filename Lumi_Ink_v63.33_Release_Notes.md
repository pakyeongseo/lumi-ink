# Lumi Ink v63.33 Release Notes

## 핵심 변경

- 아이디어 보드의 메모지 조각과 이미지 조각에 `좌우 반전` 옵션을 추가했습니다.
- 메모지는 디자인만 좌우로 뒤집히고, 안쪽 글자와 편집 도구는 정상 방향을 유지합니다.
- 이미지는 이미지 내용이 좌우 반전되어 보입니다.
- 반전 상태는 저장 데이터, 미리보기, 꾸며진 HTML 내보내기에 함께 반영됩니다.
- 새 CSS/JS가 즉시 갱신되도록 서비스워커 캐시명을 `ink-memo-v63.33-flip-items`로 변경했습니다.

## 검수 항목

- `node --check app.js` 통과
- 메모지/이미지 조각 정규화 데이터에 `flipX` 저장 확인
- 보드 렌더링, 조각 옵션 팝업, 미리보기, HTML 내보내기 클래스 반영 확인
