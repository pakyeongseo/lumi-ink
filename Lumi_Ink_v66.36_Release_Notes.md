# Lumi Ink v66.36 변경 이력

## 정규식 작업실
- Sample 입력 영역에서 완성된 <code>[[유동적 내용]]</code> 표기를 색상 박스 토큰으로 시각화했습니다.
- 토큰에는 대괄호 <code>[[ ]]</code>를 포함한 원문 전체가 들어가므로, 고정 텍스트와 유동 입력값의 경계가 즉시 보입니다.
- 이 표시는 입력 전용 오버레이이며, 저장되는 Sample 원문·복사/붙여넣기·<code>샘플→IN</code> 패턴 생성·미리보기 로직은 변경하지 않습니다.
- Sample 스크롤 시 시각 토큰도 본문과 함께 동기화됩니다.

## 배포
- 서비스 워커 캐시를 v66.36으로 갱신했습니다.
