# Lumi Ink v64 변경사항

## 페르소나/캐릭터 카드 정리

- 페르소나와 캐릭터를 서로 다른 메모 타입으로 유지하면서 같은 카드 편집기를 공유하도록 정리했습니다.
- 페르소나끼리 병합하면 페르소나 모음으로, 캐릭터끼리 병합하면 캐릭터 모음으로 생성됩니다.
- 단일 캐릭터 타입을 새 메모 타입 선택 메뉴에서 만들 수 있습니다.
- 페르소나/캐릭터 메모 메뉴에 타입 전환 기능을 추가했습니다. 단일과 모음 형태 모두 전환할 수 있습니다.
- 페르소나/캐릭터 모음에서 현재 페이지만 단일 메모로 분리 복사할 수 있습니다.

## 아이디어 보드

- 자유 색상이 어둡고 탁하게 보이던 보정값을 밝고 선명하게 조정했습니다.
- 메모지 프레임이 안쪽으로만 들어가 여백이 바깥으로 삐져나와 보이던 문제를 줄이도록 프레임 기준선을 중앙 정렬했습니다.

## HTML 작업실

- 미리보기를 새 웹페이지로 여는 버튼을 추가했습니다.
- 새로고침 버튼 아이콘의 위치와 정렬을 정리했습니다.

## 프로젝트 카테고리

- 프로젝트 내 카테고리 표시 순서를 `페르소나 - 캐릭터 - 로그 저장 - 로어북 - HTML 작업실 - 아이디어 보드 - 자유 메모` 순서로 재정렬했습니다.
- 프로젝트 화면의 다인 페르소나/캐릭터 칩 설명을 `인원수 · 타입` 형식으로 정리했습니다.
- 새 메모 팝업의 페르소나, 캐릭터, 로그 저장, 아이디어 보드 설명 문구와 아이디어 보드 아이콘을 조정했습니다.
