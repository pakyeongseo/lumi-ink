# 개발 빌드 v66.16 · 일반 JSON 열기 안정화

- 사이드바 **열기**에서 일반 `.json`을 더 안정적으로 인식하도록 파일명 확장자, 표준 JSON MIME, `+json` MIME, 실제 원문 형식을 함께 판별합니다.
- 모바일 문서 제공자가 MIME을 비우거나 `application/octet-stream`으로 넘겨도, 내용이 `{` 또는 `[`로 시작하는 JSON이면 **JSON 열기** 흐름으로 연결합니다.
- 문법 오류가 있는 JSON도 원본 작업실에서 점검할 수 있도록 유지했습니다.
- Android 계열 파일 선택기에서 패널 종료와 파일 읽기가 경합할 수 있는 흐름을 분리해, 선택한 파일 객체를 확보한 다음 프레임에 읽습니다.
- 파일 읽기 실패 시 원인명을 함께 표시하도록 오류 안내를 보강했습니다.
