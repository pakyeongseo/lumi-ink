# Lumi Ink v63.22.1 변경 요약

기준: 2026-06-24 · 아이디어 보드 안정화 및 문서 동기화 패치

## 조작 안정성
- 메모지 드래그 중 포인터 캡처가 끊기거나 종료 이벤트가 누락될 때 상태를 정리하도록 보강했습니다.
- 길게 누르기·더블탭·선택 핸들의 이벤트 경로를 분리해, 점선만 남고 옵션/이동이 반응하지 않던 상황을 줄였습니다.

## 메모지와 리치 서식
- 인앱 용어를 `포스트잇`에서 `메모지`로 통일했습니다.
- 메모지 미니 리치 에디터에 하늘색·핑크·연두·연보라·노랑의 5색 형광펜 선택을 넣었습니다.
- 리치 도구막대는 선택 핸들을 가리지 않도록 메모지 아래쪽에 배치됩니다.
- 중앙 꼬리와 내부 라인이 있는 말풍선 메모지를 다듬고, 더 둥근 `동글 말풍선 메모지`를 추가했습니다.

## 색상 입력
- 아이디어 보드의 자유 색상 선택과 자유 메모 글자색 선택에 RGB(R/G/B) 숫자 입력을 추가했습니다.
- HEX, 시스템 색상 선택기, 기본 팔레트, 저장한 색은 기존처럼 함께 사용할 수 있습니다.

## 링크·첨부·음악
- 링크는 끝단 화살표 버튼을, 첨부는 끝단 다운로드 버튼을 눌러야만 실제 동작합니다. 조각의 나머지 면적은 선택·이동에 사용됩니다.
- 음악 플레이어는 제목 표시 on/off와 보드 전용 제목 변경을 지원합니다. 원본 파일명은 변경하지 않습니다.
- 음악·링크·첨부는 조각을 크게 조절할 때 아이콘·글자·버튼도 함께 커집니다.

## 이미지·영상·배경
- 이미지와 영상 조각에 현재 프로젝트 썸네일의 프레임·색상을 적용하거나 해제할 수 있습니다.
- 투명 PNG/GIF는 컨테이너의 사각 그림자 없이 이미지 윤곽 그림자만 유지합니다.
- 제공 이미지 배경은 선택 전에도 고래 일러스트, 업로드 이미지 배경은 현재 테마색과 업로드 마크를 고정 표시합니다.

## 문서와 배포
- 전체 기능 매뉴얼을 v63.22.1 기준으로 갱신했습니다.
- 서비스 워커 캐시 키를 갱신해 설치형 앱에서 최신 매뉴얼과 정적 자산을 받을 수 있게 했습니다.
