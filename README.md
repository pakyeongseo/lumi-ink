# Lumi Ink v63.35 — Idea Board 5 Theme Templates

루미잉크는 프로젝트 단위로 여러 종류의 메모를 관리하는 로컬 웹앱/PWA입니다. 이 패키지는 배포 전 정리본이며, 현재 앱 본체와 최신 매뉴얼, 최신 릴리즈 노트를 기준 파일로 유지합니다.

## 현재 버전

- 버전: v63.35
- 릴리즈 노트: `Lumi_Ink_v63.35_Release_Notes.md`
- 상세 매뉴얼: `Lumi_Ink_Manual_1.html`
- 서비스워커 캐시: `ink-memo-v63.35-idea-board-5themes`

## v63.35 주요 변경

### 아이디어 보드 — 테마 템플릿 5종 추가

- 아이디어 보드 배경 템플릿 5종을 추가했습니다.
  - 베르사유 벽지
  - 마녀의 숲
  - 요정의 반짝임
  - 마법소녀
  - 청춘 카드
- 아이디어 보드 메모지 템플릿 5종을 추가했습니다.
  - 베르사유 카드
  - 마녀의 약초지
  - 요정 반짝 메모
  - 마법소녀 카드
  - 청춘 카드
- 템플릿 레지스트리와 CSS 선택자 매칭을 검증했습니다.

## 포함된 주요 기능

### 메모 타입

- 자유 메모
- HTML 작업실
- 로어북
- 통합 정보 카드
- 로그 저장 메모
- 아이디어 보드

### 로그 저장 메모

- 기본 제공 로그 템플릿 50종
- 사용자 템플릿 업로드
- 템플릿 즐겨찾기, 검색, 정렬
- 원본 이름/별칭 가림 처리
- 보기 모드와 원본 편집 모드 분리
- 인라인 스타일 HTML 복사 및 꾸며진 HTML 파일 내보내기

### 아이디어 보드

- 큰 캔버스 위에 메모지, 이미지/GIF, 음악/오디오, 동영상, 파일, 메모 링크 배치
- 조각별 이동, 크기 조절, 회전, 복제, 앞뒤 순서 조정
- 조각별 그림자 온/오프
- 메모지 리치 서식과 하이라이트
- 조각별 프레임과 프레임 색상
- 메모지/이미지 좌우 반전
- 음악 플레이어 다크/라이트 바 모드
- 보드 배경 템플릿, 컬러 캔버스, 제공 이미지 배경, 업로드 이미지 배경
- 보드 배경 오버레이 컬러/투명도 조절
- 화면 맞춤 줌과 직접 줌 입력
- 꾸며진 HTML 내보내기 및 아이디어 보드 HTML 가져오기

### 프로젝트 관리

- 프로젝트 단위 내보내기/불러오기
- 개별 메모와 첨부 파일 포함 백업
- 자동 백업
- 프로젝트별 아이콘, 프레임, 색상 설정

## 정리된 과거 파일

배포본을 가볍고 명확하게 유지하기 위해 다음 파일은 제거 대상입니다.

- 과거 `Lumi_Ink_Manual_v63.xx_current.html` 사본
- v63.22.1~v63.34 개별 릴리즈 노트
- 개발 확인용 `frame-demo.html`

과거 세부 변경 이력은 개발 보관본에서 관리하고, 배포 ZIP에는 최신 매뉴얼과 최신 릴리즈 노트만 포함하는 기준입니다.
