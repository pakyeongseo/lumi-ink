# Lumi Ink v63.32 — Music Player Polish

루미잉크는 프로젝트 단위로 여러 종류의 메모를 관리하는 로컬 웹앱/PWA입니다.  
이 패키지는 배포 전 정리본으로, 현재 앱 본체와 최신 매뉴얼, 최신 릴리즈 노트만 기준 파일로 유지합니다.

## 현재 버전

- 버전: v63.32
- 릴리즈 노트: `Lumi_Ink_v63.32_Release_Notes.md`
- 앱 내 매뉴얼: `Lumi_Ink_Manual_1.html`
- 서비스워커 캐시: `ink-memo-v63.32-music-player-clean`

## v63.32 주요 변경

### 아이디어 보드 — 뮤직 플레이어 디자인 개선

- 라운드 사각형 형태는 유지하면서 카드형 음악 플레이어의 색감과 깊이감을 강화했습니다.
- 흰색으로 튀던 네이티브 오디오 컨트롤을 어두운 톤으로 맞춰 카드와 더 자연스럽게 어울리도록 조정했습니다.
- 상단 음표 아이콘을 유리 질감의 둥근 배지 안에 정리했습니다.
- 선택한 조각 컬러가 플레이어 그라데이션, 외곽 글로우, 내부 림에 더 또렷하게 반영됩니다.
- 제목 숨김 상태에서도 `MUSIC` 레이블과 장식은 유지됩니다.
- 꾸며진 HTML 내보내기에서도 앱 안의 음악 플레이어와 같은 구조와 스타일이 포함됩니다.

## 포함된 주요 기능

### 메모 타입

- 자유 메모
- HTML 작업실
- 로어북
- 통합 인물 카드
- 로그 저장 메모
- 아이디어 보드

### 로그 저장 메모

- 기본 제공 로그 템플릿 50종
- 사용자 템플릿 업로드
- 템플릿 즐겨찾기, 검색, 정렬
- 원본 이름/별칭 가림 처리
- 보기 모드와 원본 편집 모드 분리
- 인라인 스타일 HTML 복사 및 꾸며진 HTML 파일 내보내기

### 아이디어 보드

- 대형 캔버스 위에 메모지, 이미지/GIF, 음악/오디오, 동영상, 파일, 메모 링크 배치
- 조각별 이동, 크기 조절, 회전, 복제, 앞/뒤 순서 조정
- 조각별 그림자 온/오프
- 메모지 리치 서식과 형광펜
- 조각별 프레임과 프레임 색상
- 보드 배경 템플릿, 컬러 캔버스, 제공 이미지 배경, 업로드 이미지 배경
- 보드 배경 오버레이 컬러/투명도 조절
- 화면 맞춤 줌과 직접 줌 입력
- 꾸며진 HTML 내보내기 및 아이디어 보드 HTML 가져오기

### 프로젝트 관리

- 프로젝트 단위 내보내기/불러오기
- 개별 메모 및 첨부 파일 포함 백업
- 자동 백업
- 프로젝트별 아이콘/프레임/색상 설정

- 핵심 JavaScript 문법 검사 통과

## 정리된 과거 파일

배포본을 가볍고 명확하게 유지하기 위해 다음 파일은 제거 대상입니다.

- 과거 `Lumi_Ink_Manual_v63.xx_current.html` 사본
- v63.22.1~v63.31 개별 릴리즈 노트
- 개발 확인용 `frame-demo.html`

과거 세부 변경 이력은 개발 보관본에서 관리하고, 배포 ZIP에는 최신 매뉴얼과 최신 릴리즈 노트만 포함하는 기준입니다.
