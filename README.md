# Lumi Ink v63.40 — Board Interaction & Replacement

루미잉크는 프로젝트 단위로 여러 종류의 메모를 관리하는 로컬 웹앱/PWA입니다. 이 패키지는 배포 전 정리본이며, 현재 앱 본체와 최신 매뉴얼, 최신 릴리즈 노트를 기준 파일로 유지합니다.

## 현재 버전

- 버전: v63.40
- 릴리즈 노트: `Lumi_Ink_v63.40_Release_Notes.md`
- 상세 매뉴얼: `Lumi_Ink_Manual_1.html`
- 서비스워커 캐시: `ink-memo-v63.40-board-operations-replace`


## v63.40 주요 변경

- 아이디어 보드 스냅 리사이즈 화면 갱신과 그룹 리사이즈 제스처를 안정화했습니다.
- 그룹 조각은 다중선택에서 하나의 선택 단위로 동작합니다.
- 메모 링크 프로젝트 목록에 썸네일과 하단 뒤로가기 흐름을 추가했습니다.
- 빈 프레임을 제목 없는 장식 조각으로 정리하고, 프레임 종류/색상은 하단 `테마` 섹션에서 선택합니다.
- 이미지·음악·동영상·첨부·메모 링크 교체 기능을 추가했습니다.
- 홈 프로젝트 고정은 최대 4개입니다.

## v63.39 주요 변경

### 아이디어 보드 — 다중선택·잠금 표시·메모지 서식 제스처

- 다중선택 모드의 조각 터치가 단일선택으로 빠지던 문제를 수정했습니다.
  - 다중선택 전환 시 조각의 포인터 동작을 다중선택 전용으로 다시 바인드합니다.
  - 조각을 터치해 선택/해제하고, 빈 캔버스 드래그로 사각 선택도 할 수 있습니다.
- 하단 다중선택 툴팁을 여섯 버튼으로 재구성했습니다.
  - 전체선택/전체해제, 정렬, 그룹, 잠금/해제, 선택 복제, 선택 삭제
  - 정렬 버튼은 왼쪽·가운데·오른쪽·위·세로 중앙·아래와 가로/세로 간격맞춤을 위쪽 패널로 펼칩니다.
- 잠긴 조각은 평소 자물쇠·점선이 보이지 않고, 선택했을 때만 표시됩니다.
- 메모지 글자색 버튼은 짧게 누르면 현재 색을 적용하고, 길게 누르면 색상 선택창을 엽니다. 형광펜도 같은 안정적인 긴 누르기 처리로 정리했습니다.
- 스냅 토스트를 `스냅 ON` / `스냅 OFF`로 간결하게 바꿨습니다.

## v63.38 주요 변경

### 아이디어 보드 — 내 메모 링크 선택기/보기 토글/스냅 저장

- 내 메모 링크 선택 팝업을 로그 템플릿 선택기 스타일로 개편했습니다.
  - `전체보기` 탭과 `프로젝트별` 탭을 제공합니다.
  - 프로젝트별 탭에서는 프로젝트 목록에 들어간 뒤 해당 프로젝트의 메모를 선택합니다.
- 아이디어 보드 보기 전용 모드를 최상단 눈/연필 아이콘으로 전환하도록 변경했습니다.
  - 보드/아티팩트 탭은 편집 화면 안의 표시 방식 전환으로 유지합니다.
- 보드별 스냅 선택 상태를 저장합니다.
  - 보드마다 스냅 켜짐/꺼짐 상태가 따로 복원됩니다.

## 포함된 주요 기능

### 메모 타입

- 자유 메모
- HTML 작업실
- 로어북
- 통합 정보 카드
- 로그 저장 메모
- 아이디어 보드

### 로그 저장 메모

- 기본 제공 로그 템플릿 50종
- 사용자 템플릿 업로드
- 템플릿 즐겨찾기, 검색, 정렬
- 원본 이름/별칭 가림 처리
- 보기 모드와 원본 편집 모드 분리
- 인라인 스타일 HTML 복사 및 꾸며진 HTML 파일 내보내기

### 아이디어 보드

- 큰 캔버스 위에 메모지, 이미지/GIF, 음악/오디오, 동영상, 파일, 메모 링크 배치
- 조각별 이동, 크기 조절, 회전, 복제, 앞뒤 순서 조정
- 조각별 그림자 온/오프
- 메모지 리치 서식과 하이라이트
- 조각별 프레임과 프레임 색상
- 메모지/이미지 좌우 반전
- 음악 플레이어 다크/라이트 바 모드
- 보드 배경 템플릿, 컬러 캔버스, 제공 이미지 배경, 업로드 이미지 배경
- 보드 배경 오버레이 컬러/투명도 조절
- 화면 맞춤 줌과 직접 줌 입력
- 꾸며진 HTML 내보내기 및 아이디어 보드 HTML 가져오기

### 프로젝트 관리

- 프로젝트 단위 내보내기/불러오기
- 개별 메모와 첨부 파일 포함 백업
- 자동 백업
- 프로젝트별 아이콘, 프레임, 색상 설정
