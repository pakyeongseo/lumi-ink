# Lumi Ink (루미잉크)

AI 롤플레이 창작을 위한 **올인원 메모 · 자료 관리 PWA**.
프로젝트(폴더)별로 자유 메모 · 로어북 · 페르소나 카드를 정리하고, SillyTavern으로 바로 내보낼 수 있어요. 모든 데이터는 기기 안(IndexedDB)에만 저장되며 서버로 전송되지 않습니다.

---

## ✨ 주요 기능

### 프로젝트 (폴더)
- 메모를 프로젝트 단위로 묶어 관리
- 폴더 아이콘 이미지, 색상 태그, **태그 칩**(쉼표 구분) 지정
- 상단 고정(최대 6개), 정렬(최신순 / 오래된순 / 이름순)

### 자유 메모
- 서식 보존 리치 에디터(굵게·기울임·밑줄·취소선, 색상/형광펜, 정렬, 글자 크기, 목록)
- 이미지 삽입, 파일 첨부, **코드 블록**, **하이퍼링크**
- 지우개 버튼으로 서식·코드블록·링크를 한 번에 제거
- 읽기 화면 코드블록 복사 버튼, 본문 길게 눌러 전체 복사

### 로어북 (World Info)
- 마크다운 작성 + 보기 모드(제목 칩 박스, `- ` 삼각형 목록, `<태그></태그>` 박스, `- 라벨:` 미니 칩)
- 발동 키워드, 항상 활성(constant), **삽입 깊이(Depth)** 설정
- 실시간 토큰 카운터(cl100k)
- **SillyTavern World Info JSON** 내보내기 / 불러오기(여러 개 묶음 지원)

### 페르소나 카드
- 포트레이트(3:4) · 정사각 썸네일 · 갤러리(멀티 업로드 + 드래그 정렬)
- 한국어 / 영어 이중 양식, 태그, 마크다운 상세
- `<user name="{{user}}">` 류 제어 태그는 보기 화면에서 자동 숨김
- 토큰 카운터
- **HTML로 내보내기**(보기 양식 그대로) → 다시 "열기"로 페르소나 메모로 불러오기 가능(자료 이전/공유용)

### 공통
- 검색(제목 / 내용 / 제목+내용), 다중 선택(이동·삭제), 상단 고정
- 라이트 / 다크 테마, 웹폰트 전환
- 수동 저장 버튼 + **Ctrl/⌘ + S** 단축키(자동 저장과 별개로 즉시 저장)
- 전체 백업(.html) / 복원, 데이터 초기화

## 📲 설치 방법

웹앱(PWA)이라 설치가 매우 간단합니다.

### Android (Chrome)
1. 배포된 주소를 Chrome으로 엽니다.
2. 우측 상단 ⋮ 메뉴 → **앱 설치**(또는 "홈 화면에 추가").
3. 홈 화면 아이콘으로 실행하면 전체화면 앱처럼 동작합니다.

### iOS (Safari)
1. Safari로 주소를 엽니다.
2. 공유 버튼 → **홈 화면에 추가**.

### 데스크톱 (Chrome / Edge)
1. 주소창 오른쪽의 **설치** 아이콘 클릭, 또는 메뉴 → 앱 설치.

> 오프라인에서도 동작합니다. 업데이트는 온라인 상태에서 앱을 다시 열면 자동 적용돼요(필요 시 새로고침).

## 🚀 직접 호스팅 (다운로드 후 사용)

정적 파일만으로 동작하므로 어디서든 호스팅할 수 있습니다.

1. `ink-memo-pwa` 폴더 전체를 받습니다.
2. 웹 서버에 올립니다. 예시(로컬 테스트):
   ```bash
   cd ink-memo-pwa
   python3 -m http.server 8080
   # 브라우저에서 http://localhost:8080
   ```
   GitHub Pages 같은 정적 호스팅의 하위 폴더에 그대로 올려도 됩니다(모든 경로가 상대경로).
3. 처음 접속하면 서비스워커가 캐시를 구성하고, 이후 오프라인에서도 열립니다.

> 모든 데이터는 브라우저(IndexedDB)에 저장됩니다. 기기/브라우저를 옮길 때는 **설정 → 전체 백업**으로 `.html` 파일을 만들어 새 기기에서 **복원**하세요.

## 🔗 내보내기 요약
| 종류 | 형식 | 용도 |
|---|---|---|
| 로어북 | World Info `.json` | SillyTavern 가져오기 · 내보내기 |

| 전체 | 백업 `.html` | 기기 이전 / 복원 |
