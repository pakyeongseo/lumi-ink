# Lumi Ink v65.8 Release Notes

- **설치 페이지 리뉴얼**: 컬러별 설치 페이지를 새 디자인으로 교체하고, `lumi-ink-get-started.html` 간판 페이지를 패키지 및 오프라인 캐시에 포함했습니다. 설정의 **앱 설치 아이콘**도 간판 페이지로 연결됩니다.
- **퀵 메뉴 기능 바로가기**: 기존 홈 바로가기를 확장한 기능 바로가기를 추가했습니다. 홈·설정·설정 화면의 테마/컬러/폰트/툴바/퀵메뉴/설치/백업/복원/자동백업/저장공간/초기화/매뉴얼을 슬롯으로 지정할 수 있습니다. 기존 홈 슬롯은 자동으로 홈 기능 바로가기로 이전됩니다.
- **사용자 지정 테마 독립 선택**: 사용자 지정을 선택하면 프리셋 선택이 해제되고, 라이트/다크 모드 각각의 두 색 그라데이션 팔레트를 독립적으로 적용합니다. 모드별 ‘사용’ 체크박스는 제거했습니다. 프리셋을 다시 선택하면 사용자 지정 팔레트는 보관되지만 적용은 중단됩니다.

사용자 배포 표기는 v1.5를 유지하며, 이 문서는 개발 기준 v65.8 변경점입니다.
