# Lumi Ink v63.23 — 아이디어 보드 프레임 라이브러리·9-slice 확장

## 이번 변경

- 이미지·영상 조각의 프레임 설정을 프로젝트 썸네일과 동일한 전체 라이브러리로 확장했습니다.
  - 클래식, 비드, 바로크, 아르데코, 로프, 고딕, 보태니컬, 스캘럽, 스타라이트, 로열과 슬림 프레임 전체
  - 골드부터 파스텔·글래스·글래시한 명암·테마 연동까지 동일한 색상 선택 환경
- 프레임은 조각별로 독립 선택할 수 있습니다. 한 보드 안에서 서로 다른 프레임·색상을 함께 사용할 수 있습니다.
- ‘프로젝트와 연동’을 선택하면 프로젝트 썸네일 프레임을 바꿀 때 해당 이미지·영상 조각도 함께 갱신됩니다.
- 프레임을 조각의 바깥선에 걸쳐 배치했습니다.
- 가로·세로 리사이즈에는 9-slice 렌더링을 적용했습니다. 네 모서리와 장식은 고정되고, 위·아래·좌·우 중앙부만 늘어나므로 비율이 달라도 프레임 디자인이 뭉개지지 않습니다.
- 꾸며진 아이디어 보드 HTML 내보내기에서도 선택된 프레임을 재현합니다.

## 호환성

- v63.22.1에서 ‘프로젝트 프레임 사용’을 켠 기존 이미지·영상 조각은 자동으로 **프로젝트와 연동** 모드로 이어집니다.
- 기존 보드·프로젝트·첨부 파일은 수정하거나 삭제하지 않습니다.
