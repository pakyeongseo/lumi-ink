# Lumi Ink v64.2 변경사항

## 새 메모 생성기 — 작업 영역별 재구성

새 메모 팝업을 단순한 긴 목록에서 **캐릭터 / 작업실 / 아뜰리에**의 세 작업 영역으로 재구성했습니다.

- **상단 빠른 생성 버튼** — 현재 테마색을 따라 바뀌는 그라데이션 **「그라데이션 자유 메모」** 버튼을 추가했습니다. 자유 메모를 선택할 필요 없이 바로 생성 흐름으로 들어갑니다.
- **캐릭터** — 페르소나, 다인 페르소나, 캐릭터, 다인 캐릭터를 한 탭에서 구분해 선택합니다.
  - 기존 `단일 캐릭터`는 **캐릭터**로 이름을 정리했습니다.
  - 기존 `캐릭터`(여러 명 묶음)는 **다인 캐릭터**로 이름을 정리했습니다.
- **작업실** — 로어북, HTML 작업실.
- **아뜰리에** — 자유 메모, 로그 저장, 아이디어 보드.
- 기존 카드 데이터 형식과 단일/다인 모드는 유지합니다. 이번 변경은 생성 흐름과 표기 체계의 개선입니다.

## 자동 백업 — 보관 수 직접 설정

설정 → 자동 백업에서 보관할 스냅샷 수를 직접 조절할 수 있습니다.

- **최소 1개 / 최대 15개 / 기본 10개**
- 설정 화면과 저장공간 요약에 현재 보관 설정을 표시합니다.
- 보관 수를 줄이는 즉시 오래된 자동 백업부터 정리합니다.
- 자동 백업 목록과 복원 기능은 그대로 유지하며, 관리 창 안에서 바로 목록으로 이동할 수 있습니다.
