# Lumi Ink v63.39 — 다중선택·잠금 표시·메모지 색상 제스처

## 아이디어 보드

### 다중선택 툴바 재구성

하단 툴바를 다음 6개 주 버튼으로 정리했습니다.

1. **전체선택** — 모든 조각을 선택하며, 모두 선택된 상태에서 다시 누르면 전체 선택을 해제합니다.
2. **정렬** — 버튼 위로 정렬 패널이 펼쳐집니다.
   - 왼쪽 / 가로 중앙 / 오른쪽
   - 위 / 세로 중앙 / 아래
   - 가로 간격맞춤 / 세로 간격맞춤
3. **그룹** — 그룹화 / 그룹 해제 선택지를 펼칩니다.
4. **잠금 / 잠금 해제** — 선택 조각의 잠금 상태를 일괄 전환합니다.
5. **선택 복제** — 선택 조각을 24px씩 비껴 복제하며 새 복제본은 잠금이 해제됩니다.
6. **선택 삭제** — 잠기지 않은 선택 조각을 확인 후 일괄 삭제합니다.

다중선택 모드를 켤 때 조각의 포인터 이벤트를 다시 연결하도록 수정했습니다. 이제 조각을 터치하면 단일 선택으로 전환되지 않고 선택/해제가 누적됩니다.

### 잠금 표시

잠긴 조각의 자물쇠 배지와 점선 경계는 평상시 숨깁니다. 단일 선택·다중 선택·그룹 선택 상태일 때만 나타납니다. 잠긴 조각은 선택은 가능하지만 정렬·간격맞춤·그룹화에서는 제외됩니다.

### 메모지 미니 리치 에디터

- **글자색**: 짧은 터치는 현재 글자색을 적용합니다. 길게 누르면 색상 선택창을 엽니다.
- **형광펜**: 짧은 터치는 현재 형광펜 색을 적용합니다. 길게 누르면 다섯 색 선택창을 엽니다.
- 색상창은 긴 누르기 도중이 아니라 손을 뗀 뒤 열리도록 바꿔, 팝업이 열리자마자 같은 터치로 닫히는 문제를 막았습니다.

### 스냅 안내

스냅 토스트 문구를 `스냅 ON` / `스냅 OFF`로 단순화했습니다.
