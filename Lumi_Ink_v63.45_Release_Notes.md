# Lumi Ink v63.45 Release Notes

기준 빌드: v63.44 integrated export/divider

이번 빌드는 아이디어 보드에서 실제 사용 조건에서 남아 있던 이벤트, 잠금, 색상, 썸네일, 스크린샷 문제를 보수한 버전입니다.

## 수정 사항

- 다중선택 모드에서 조각 터치가 선택/해제로 누적되지 않던 문제를 보완했습니다.
  - `pointerdown` 외에 `touchstart`, `click` fallback을 추가했습니다.
  - 중복 이벤트로 두 번 토글되지 않도록 시간 가드를 넣었습니다.
- 그룹화 시 잠금된 조각은 그룹에서 제외되도록 명확히 처리했습니다.
  - 잠금되지 않은 조각이 2개 이상일 때만 그룹화합니다.
  - 잠금 조각이 제외되면 별도 안내를 표시합니다.
- 메모지 미니 리치 에디터의 글자색 적용을 수정했습니다.
  - 선택한 Range만 직접 `span style="color:..."`로 감쌉니다.
  - 색상 선택 시 메모지 기본 글자색(`textColor`)을 바꾸지 않아 전체 본문이 함께 바뀌지 않습니다.
- 색상 선택기 스포이드 동작을 보완했습니다.
  - 지원 브라우저 판단을 완화했습니다.
  - 스포이드 성공/미지원/실패 상태를 토스트로 안내합니다.
- 내 메모 링크의 프로젝트 탭 썸네일 프레임을 홈/사이드바 프로젝트 썸네일과 같은 외곽 프레임 구조로 보정했습니다.
- 잠금 메모지에서 `Ctrl+A`를 눌렀을 때 조각 메뉴가 뜨지 않고 본문 전체 선택이 되도록 예외 처리했습니다.
- 잠금 상태의 선택된 메모지 툴팁과 조작 핸들이 다른 조각 아래로 깔리지 않도록 레이어 우선순위를 높였습니다.
- 보기 모드 현재 화면 스크린샷 저장을 보완했습니다.
  - SVG `foreignObject`를 문자열이 아니라 DOM으로 구성한 뒤 직렬화합니다.
  - PNG 변환이 브라우저 제한으로 실패하면 SVG 스냅샷으로 대체 저장합니다.

## 검증

- 서비스워커 캐시 파일 존재 확인
- manifest JSON 파싱 확인
- HTML 중복 ID 확인
