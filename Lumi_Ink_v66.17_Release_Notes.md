# 개발 빌드 v66.17 · Android 일반 JSON 선택 필터 수정

- 사이드바 **열기**의 파일 선택기 `accept` 필터를 제거했습니다. Android 문서 앱이 `.json` 파일에 비표준 MIME을 붙여 회색 비활성화하던 문제를 해결합니다.
- 이제 파일 선택기에서는 모든 파일이 보이며, 앱 내부에서 HTML · JSON · 일반 텍스트를 판별합니다.
- 확장자 또는 MIME이 불완전해도 파일 내용이 JSON이면 **JSON 열기** 선택 창으로 연결합니다.
- NUL 문자나 제어문자가 많은 이진 파일은 앱 안에서 차단하고 지원 형식을 안내합니다.
