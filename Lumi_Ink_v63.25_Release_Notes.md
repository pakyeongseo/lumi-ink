# Lumi Ink v63.25 — 동영상 플레이어·자유 메모 인라인 영상

## 아이디어 보드

- 동영상 조각의 어두운 배경 매트를 없애고, 테마 컬러에 반응하는 밝은 카드형 플레이어 쉘로 변경했습니다.
- 재생 상태를 직관적으로 알 수 있도록 작은 `PLAY` 배지와 별 장식을 추가했습니다. 브라우저 기본 재생·음량·전체화면 컨트롤은 그대로 유지합니다.
- 음악 플레이어에서 제목 표시를 꺼도 상단이 사라지지 않습니다. 레코드 장식, 확대된 `MUSIC` 레이블, 별 장식은 유지하고 제목 텍스트만 숨깁니다.

## 자유 메모

- 툴바의 미디어 버튼을 누르면 `사진` 또는 `동영상`을 선택할 수 있습니다. 두 항목 모두 일반 파일선택기를 사용합니다.
- 동영상은 MP4/WebM/Ogg/MOV 형식, 30MB 이하에서 본문에 인라인 삽입됩니다.
- 삽입된 동영상은 사진과 같은 선택 테두리, 모서리 드래그, 가로·세로 픽셀 입력, 비율 고정, 자동 크기 복원을 지원합니다.
- 동영상은 앱이 생성한 안전한 `data:video`만 허용하도록 정화 규칙을 확장했습니다. 외부 원격 영상·스크립트성 미디어는 허용하지 않습니다.
- 전체 백업/복원과 일반 HTML 내보내기·재가져오기에서 인라인 동영상이 유지됩니다.
