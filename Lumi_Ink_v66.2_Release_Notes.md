# Lumi Ink v66.2 — 로그 세트 중복 차단 · 퀵메뉴 아이콘 균형

- 로그 이름 치환 세트의 원본 이름은 다섯 세트 전체에서 중복 저장할 수 없도록 변경했습니다.
  - 입력 중 다른 세트와 겹치는 이름을 즉시 안내합니다.
  - 충돌이 있으면 저장 버튼이 비활성화되며, “그래도 저장” 경로는 제거했습니다.
  - 이전 데이터에 있던 중복은 낮은 번호 세트(1 → 5)를 우선 보존하고 뒤 세트에서 안전하게 제외합니다.
- 로그 상단의 다섯 세트 버튼을 간결한 `세트 1`~`세트 5` 타일로 재구성했습니다.
  - 설정 완료 여부는 작은 상태 점으로만 표시합니다.
  - 모든 화면 폭에서 다섯 버튼이 한 줄로 유지됩니다.
- 퀵메뉴 내장 아이콘의 그림 확대값을 v65.9 기준 80%로 조정했습니다.
  - 미디어 박스 규격은 유지하면서 프로젝트 썸네일과 더 균형 잡힌 여백을 사용합니다.
