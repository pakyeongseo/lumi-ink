# Lumi Ink v63.36 Release Notes

## 핵심 변경

- 아이디어 보드에 `보기` 전용 모드를 추가했습니다.
  - 보드/아티팩트/보기 탭으로 전환합니다.
  - 보기 모드에서는 조각 선택, 이동, 크기 조정, 회전, 삭제, 추가, 스냅 토글 같은 편집 동작이 막힙니다.
  - 메모 링크 열기, 첨부 다운로드, 오디오/동영상 재생 같은 보기 동작은 유지됩니다.
- 내 메모 링크 조각의 하단 표시를 내용 미리보기 대신 메모 종류로 고정했습니다.
  - 예: 자유 메모는 `자유메모`로 표시됩니다.
- 내 메모 링크 선택 팝업에 검색 바를 추가했습니다.
  - 제목, 내용, 메모 종류를 기준으로 필터링합니다.
- 메모지, 음악, 첨부, 링크 조각의 크기 직접 입력 창에도 `비율 고정` 옵션을 추가했습니다.
  - 이미지/동영상은 기존처럼 기본 비율 고정입니다.
  - 메모지/음악/첨부/링크는 기본 비율 고정 해제입니다.
- 메모지를 감싸는 프로젝트 프레임 위치를 보정했습니다.
  - 프레임의 중앙 라인이 메모지 외곽선에 더 가깝게 맞도록 조정했습니다.

- `node --check service-worker.js` 통과

- 서비스워커 캐시명 갱신: `ink-memo-v63.36-idea-board-readonly`
