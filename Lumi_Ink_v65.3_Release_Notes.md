# Lumi Ink v65.3 — SVG Icon Library Integration

사용자 배포 버전은 **v1.5**이며, 이 문서는 개발 추적용 **v65.3** 변경 기록입니다.

## 퀵메뉴 아이콘 라이브러리
- 제공받은 SVG 아이콘 30종을 내장 아이콘 모음으로 등록했습니다.
- 슬롯 편집 화면에 **아이콘 모음** 진입점을 추가했습니다.
- 이동·기능·기록·장식 카테고리로 필터링해 고를 수 있습니다.
- 선택한 아이콘은 SVG 원본을 유지해 작은 크기에서도 선명하게 표시됩니다.
- 다수 아이콘을 동시에 렌더링할 때 SVG `defs` / 그라데이션 ID가 서로 충돌하지 않도록 인스턴스별 ID를 재작성합니다.
- 카드형 슬롯과 미니형 슬롯 모두에서 동일하게 표시됩니다.

## 프로젝트 썸네일
- 프로젝트 썸네일 팝업을 **기본 이미지 / SVG 아이콘 모음** 탭으로 분리했습니다.
- SVG 아이콘을 프로젝트 대표 썸네일로도 선택할 수 있습니다.
- 기존 기본 이미지·사용자 업로드 이미지는 그대로 유지하며, 새 탭에서만 SVG 아이콘을 선택합니다.
- 프레임 기능도 SVG 아이콘 위에 정상적으로 적용됩니다.

## 디자인 다듬기
- 원본 아이콘의 네이비·블루·골드 조합과 그라데이션은 유지했습니다.
- 퀵메뉴와 프로젝트 썸네일 컨테이너에만 미세한 입체감·정렬·그림자 보정을 적용했습니다.
- 아이콘 소스 자체를 래스터 이미지로 변환하거나 전역 테마 색상으로 재염색하지 않습니다.

## 배포
- 서비스 워커 캐시 키: `ink-memo-v1.5-v65.3-svg-icon-library`
- 새 정적 자산: `quickmenu-icon-library.js`
