# Lumi Ink v66.33 릴리스 노트

## ✨ 아이디어 보드 — 구분선 디자인 25종 추가 (총 32 → 57종)

기존 32종에 더해, 기법이 겹치지 않는 25종을 새로 등록했습니다. 모두 테마 컬러(`--idea-color-b`)와 굵기 슬라이더(`--idea-divider-weight`)에 자동 연동되며, 내보내기 HTML에도 동일하게 적용됩니다.

### 선 텍스처 8종
- **물결(wave)** · **지그재그(zigzag)** — SVG 마스크로 곡선·꺾은선을 색 변수 그대로 렌더
- **꼬임(rope)** — 두 톤 사선이 꼬인 밧줄
- **사슬(chainLink)** — 고리가 이어진 띠
- **음각(groove)** — 상암하명 입체 홈선
- **리본(ribbon)** — 굵은 중심선 + 가는 테두리줄
- **바코드(barcode)** — 폭이 불규칙한 세로 막대
- **스캘럽(scallop)** — 반원이 이어지는 부채꼴

### 점·구슬 4종
- **진주(pearlRow)** — 큰 진주와 잔점 교차
- **점강(gradedDots)** — 가운데가 굵고 양끝으로 작아지는 점선
- **그물(lattice)** — 교차 사선 격자
- **벽돌(brick)** — 위아래 두 줄이 반칸 엇갈린 블록

### 중앙 글리프 8종
반짝(❋) · 눈(❄) · 해(❂) · 네잎(✤) · 십자가(✛) · 별표(✳) · 꽃송이(❀) · 빈 마름모(◇ 윤곽)

### 양끝 장식 5종
마름모 끝 · 점 끝 · 고리 끝 · 깃 끝(❦ 대칭) · 잎 끝(❧ 대칭)

## 🔧 기술 메모
- 신규 스타일은 모두 `.idea-divider` / `.idea-preview-divider` 선택자 양쪽에 정의 — 미리보기 타일과 본 화면 일관 유지
- 기존 32종 키·라벨·CSS는 전량 보존, 추가만 수행 (기존 사용자 데이터 호환)
- 곡선 2종(물결·지그재그)은 SVG `mask` + `background:var(--c)` 방식으로 색 연동
- 캐시 버전 `v66.33-divider-pack-25`로 bump
