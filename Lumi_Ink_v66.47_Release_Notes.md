# Lumi Ink v66.47 릴리스 노트

## ✨ 아이디어 보드·자유메모 — 장르 테마 구분선 13종 추가 (총 87 → 100종)

봇 무드에 맞춰 쓸 수 있는 장르 감성 구분선 13종을 추가했습니다. 테마 컬러(`--idea-color-b`)에 연동되며, `color-mix`로 장르 무드(고딕의 음영·핏빛, 마법소녀의 파스텔·발광)를 가미했습니다.

### 🖤 고딕 호러 5종
- **가시덩굴(thorns)** — 가시 돋은 덩굴
- **박쥐떼(bats)** — 날개 편 박쥐 실루엣 반복
- **핏방울(bloodDrip)** — 핏빛 선에서 길이가 다른 방울이 흘러내림
- **거미줄(cobweb)** — 늘어진 거미줄 커튼
- **고딕 십자가(gothicCross)** — 어두운 선 + 십자가 + 링크

### 💗 마법소녀 5종
- **별가루(stardust)** — 중앙 별 + 흩날리는 별가루 발광
- **하트 체인(heartChain)** — 하트가 이어진 파스텔 띠
- **마법봉(magicWand)** — 빛나는 별 + 별가루 트레일
- **반짝이(sparkleTrail)** — 중앙 발광 다이아 + 점강 트레일
- **리본(ribbonBow)** — 중앙 리본 매듭 + 파스텔 띠

### 🌙 오컬트·판타지 3종
- **달의 위상(moonPhases)** — 초승달→보름달 반복
- **룬(runicLine)** — 직선 룬 문자 반복
- **펜타클(pentacle)** — 원 안의 오망성

## 🔧 기술 메모
- 박쥐·하트·거미줄·달위상·룬·가시는 SVG `mask` + `background:var(--c)`로 색 연동
- 발광 계열은 `box-shadow`/`filter:drop-shadow`로 글로우, `color-mix`로 다색 펄
- 핏방울은 길이가 제각각인 `radial-gradient` 다층으로 흘러내림 표현
- 기존 87종 키·라벨·CSS는 전량 보존, 추가만 수행
- 옵션 안내 문구 "디자인 87종" → "100종", 캐시 `v1.7.1-divider-genre-13`로 bump
