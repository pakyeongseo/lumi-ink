# Lumi Ink v65.9 Release Notes

- **퀵메뉴 아이콘을 미니 썸네일로 보정**: 내장 아이콘이 기본 글리프와 같은 39px(미니형 37px) 미디어 박스를 사용하면서도, SVG 내부 안전 여백만 확대해 이미지 썸네일처럼 타일을 꽉 채워 보이도록 수정했습니다. 업로드 이미지·기본 기능 아이콘의 박스 규격은 바꾸지 않았습니다.
- **그라데이션 중심 아이콘 팔레트**: 70종 내장 SVG의 색상 역할을 테마별 `--accent`, `--accent-2` 두 메인 그라데이션 색으로 다시 조합했습니다. 테마의 가장 어두운 구조색(`--ink`, `--bg`, `--accent-deep`)은 아이콘 외곽선·음영에 더 이상 사용하지 않습니다.
- **라이트/다크 공통 보정**: 밝은 하이라이트·종이색·포인트색·음영을 같은 메인색 계열 안에서 분리해, 파스텔 테마는 탁해지지 않고 딥 테마도 검은 윤곽선 없이 형태를 읽을 수 있게 조정했습니다.

사용자 배포 표기는 v1.5를 유지하며, 이 문서는 개발 기준 v65.9 변경점입니다.
