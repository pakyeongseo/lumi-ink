# Lumi Ink v63.40 — 보드 조작·프레임·링크 교체 정비

## 아이디어 보드

- 스냅을 켠 채 크기를 조절할 때 화면에서는 멈추고 저장값만 바뀌던 오류를 수정했습니다.
  - 리사이즈 제스처가 현재 보드 데이터와 스냅 가이드를 같은 프레임에서 갱신합니다.
- 그룹 조각 리사이즈 중 조절 핸들이 매 프레임 새로 만들어져 움찔거리거나 캔버스가 밀리던 문제를 수정했습니다.
  - 그룹 박스와 핸들을 유지한 채 좌표·크기만 갱신합니다.
- 다중선택에서 이미 그룹인 조각 하나를 누르면 그룹 전체가 한 선택 단위로 선택됩니다.
  - 사각 선택에서도 그룹 전체가 함께 선택됩니다.
- 메모 링크 선택기에 프로젝트 썸네일을 추가했습니다.
  - 프로젝트 안으로 들어간 뒤에는 검색창 옆 버튼을 없애고, 하단 닫기 버튼 왼쪽에 `← 뒤로가기`을 표시합니다.
- 빈 프레임은 캔버스 중앙과 목록에 표시 제목을 출력하지 않는 장식 전용 조각으로 정리했습니다.
  - 옵션창의 상단 프레임 버튼은 제거하고, 하단 `테마` 섹션 안에서 프레임 종류와 프레임 컬러를 고릅니다.
- 이미지·음악·동영상·첨부파일·메모 링크에 `교체`를 추가했습니다.
  - 새 파일/연결 메모로 바꿔도 위치, 크기, 색상, 프레임, 잠금 등의 조각 속성은 유지됩니다.
  - 기존 파일은 되돌리기 기록에서 참조하는 동안 바로 삭제되지 않습니다.

## 홈

- 홈 화면 프로젝트 고정 한도를 3개에서 **4개**로 늘렸습니다.
