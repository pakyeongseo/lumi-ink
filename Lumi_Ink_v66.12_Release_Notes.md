# Lumi Ink v66.12 · JSON 원문 파일 저장

> 사용자 배포 표기는 **v1.5**를 유지합니다.
> 개발 빌드: **v66.12**

## HTML 작업실 파일 저장

- HTML 작업실의 작업 막대에 **파일 저장** 버튼을 추가했습니다.
- 파일 저장 창에서 파일 이름과 확장자를 **`.html` / `.json`** 중 선택할 수 있습니다.
- 일반 HTML 작업실은 `.html`을, 사이드바에서 JSON 원문으로 연 작업실은 `.json`을 기본 선택합니다.
- 선택한 기본 확장자는 해당 작업실에 보관되어 다음 파일 저장 때 다시 제안됩니다.
- 저장하는 원문은 앱이 태그·공백·줄바꿈을 덧붙이거나 정화하지 않습니다. 확장자와 MIME만 선택값에 맞춰 적용합니다.

## JSON 보호

- `.json` 저장 전에는 JSON 문법을 검사합니다.
- JSON 문법 오류가 있으면 파일을 생성하지 않고, 가능한 경우 오류 위치를 안내합니다.
- `.html` 저장은 원문 형식과 관계없이 가능합니다. JSON 원문을 HTML 확장자로 보관해야 하는 특수 상황도 막지 않습니다.

## 조작

- **파일 저장:** 작업 막대의 파일 아이콘 또는 더보기 → `파일로 저장 (.html / .json)`
- **앱 내부 저장:** 기존 상단 디스켓 또는 `Ctrl/⌘+S`
- **파일 저장 단축키:** `Ctrl/⌘+Shift+S` (HTML 작업실에서만)

## 호환성

- 기존 HTML 작업실 메모는 자동으로 `.html` 기본값을 사용합니다.
- 이전 전체 백업·프로젝트 패키지를 가져와도 기존 HTML 작업실은 그대로 복원됩니다.
- JSON 원문으로 열었던 작업실의 기본 확장자 정보는 이후 전체 백업과 프로젝트 패키지에도 함께 보존됩니다.
