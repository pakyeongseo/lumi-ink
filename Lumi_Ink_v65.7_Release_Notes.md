# Lumi Ink v65.7 Release Notes

## v64.9 팔레트 기준 복원 · 두 색 사용자 지정 그라데이션

- v64.9 패키지의 기본 CSS 색상 토큰과 프리셋별 색상 배치를 기준값으로 되돌렸습니다. 기본 프리셋의 로고, 버튼, 텍스트, 프로젝트 기본 아이콘 및 그라데이션은 사용자 지정 기능이 꺼진 상태에서 원본 색을 그대로 사용합니다.
- 내장 SVG 아이콘은 라이트 모드의 연한 보조 토큰을 외곽선 색으로 잘못 사용하던 문제를 수정했습니다. 밝은 색·중간 색·딥 잉크색·하이라이트·종이색을 별도 역할로 조합해 라이트/다크 모두에서 또렷하게 렌더링합니다.
- 퀵메뉴의 내장 아이콘은 기본 아이콘/업로드 이미지와 같은 39px(미니형 37px) 미디어 박스를 사용합니다. 벡터 그림만 그 안에서 확대해 여백 없이 꽉 차게 표시합니다.
- 사용자 지정 테마의 메인 색상을 두 개의 그라데이션 스톱(시작/끝)으로 분리했습니다. 두 색을 바탕으로 라이트·다크 각각의 추천 팔레트를 생성합니다.
- 사용자 지정 테마가 활성화된 모드에서는 로고 펜촉, 별 장식, 버튼 그라데이션, SVG 아이콘이 같은 두 색의 기조를 사용합니다. 프리셋 모드에는 영향을 주지 않습니다.
- 기존 v65.6 이하의 단일 메인 색상 사용자 지정 데이터는 첫 색상으로 유지되고, 두 번째 색상은 안전한 보조 그라데이션 값으로 자동 보완됩니다.

## 버전
- 개발 버전: v65.7
- 사용자 배포 버전: v1.5
