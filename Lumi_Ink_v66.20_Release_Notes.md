# 개발 빌드 v66.20 · 사용자 지정 테마 정렬 글자 역할 통합

- 사용자 지정 컬러 테마의 **메인 화면 섹션 제목·정렬 글자** 색상 역할을 하나로 통합했습니다.
- 홈 화면의 정렬 버튼과 프로젝트 상세 화면의 정렬 버튼은 모두 `--home-section-title-color`를 사용합니다.
- 정렬 버튼의 텍스트뿐 아니라 정렬 아이콘 선색도 같은 색상 역할을 따라가므로, 제목과 버튼의 강조 결이 분리되지 않습니다.
- 정렬 버튼 배경은 기존처럼 **메인 화면 정렬 배경** / 공통 옅은 강조·선택 배경 역할을 유지합니다.
