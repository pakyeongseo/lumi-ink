# Lumi Ink v65.5 변경사항

## 퀵 메뉴 아이콘 확장 및 편집 동선 정리

- 러블리 큐티, 자연의 싱그러움, 쥬얼리 럭셔리, 모던 테마 아이콘을 내장 아이콘 모음에 추가했습니다.
- 추가된 아이콘의 모든 색상은 Lumi Ink의 아이콘 컬러 변수로 변환되어, 선택한 앱 테마의 강조색·명암에 반응합니다.
- 퀵 메뉴의 내장 SVG 아이콘 표시 비율을 키워 슬롯 안을 더 꽉 채우도록 조정했습니다.
- 슬롯 편집에서 `이미지 썸네일`을 `썸네일 업로드`로 변경하고, 즉시 파일 선택기를 엽니다.
- 아이콘/이미지/코드별 해제 버튼을 제거하고 `기본 설정` 하나로 통합했습니다. 이 버튼은 사용자 썸네일·내장 아이콘·SVG/CSS 아이콘을 모두 지우고 해당 동작의 기본 아이콘으로 되돌립니다.
- 사용자 썸네일, 내장 아이콘, 코드 아이콘은 서로 하나만 활성화되도록 정리했습니다.
- 라이트 모드 슬롯 편집 팝업의 텍스트와 중립 버튼이 검정으로 떨어지지 않고 테마 기반의 짙은 잉크색을 사용합니다.

## 배포 갱신

- 서비스 워커 캐시 키를 `ink-memo-v1.5-v65.5-themed-icons-quickmenu-refine`로 갱신했습니다.
