# Lumi Ink v63.26 — 아이디어 보드·자유메모 미디어 입력 안정화

## 아이디어 보드

- **바로크 프레임**만 9-slice 분할값을 별도로 조정했습니다. 넓은 모서리 장식은 보존하고, 중앙의 좁은 띠만 가로·세로로 늘어납니다.
- 동영상 조각의 기존 검은 매트 규칙을 명시적으로 덮어써, 밝은 파스텔 카드형 쉘·PLAY 배지·별 장식이 실제 보드에 반영됩니다.
- 메모지 형광펜은 **짧게 탭하면 현재 색상 즉시 적용**, **0.48초 이상 길게 누르면 다섯 색 선택창**을 열도록 제스처를 분리했습니다.
- 조각의 수치 크기 입력창에 **비율 고정** 체크박스를 추가했습니다. 새 조각은 기본으로 고정되며, 체크 상태도 조각마다 저장됩니다.

## 자유메모

- 사진/동영상 선택은 모두 모바일 갤러리 전용 선택기가 아닌 **범용 파일 선택기**를 통해 열립니다. 선택 후에만 파일 형식을 검증합니다.
- 새 인라인 동영상은 삽입 직후 본문 폭에 맞는 크기로 초기화됩니다. 메타데이터 로드 전 원본 크기로 튀어 핸들이 어긋나던 문제를 보정했습니다.
- 동영상을 다시 터치하면 브라우저 기본 컨트롤 위에서도 미디어 선택 테두리와 크기 조절창이 다시 표시됩니다.

## 호환성

- 기존 자유메모의 이미지·동영상과 기존 아이디어 보드 조각은 그대로 유지됩니다.
- 서비스 워커 캐시를 v63.26으로 갱신했습니다.
