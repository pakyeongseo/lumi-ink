# Lumi Ink v63.31 변경사항

## 색상 선택기 · 아이디어 보드 다듬기

- **색상 선택기 샘플 영역 제거** — 색상판 아래에 이미 선택한 색이 보이므로, 중복되던 미리보기(샘플색·샘플 글자) 영역을 없앴습니다. 화면이 더 짧고 깔끔해졌습니다.
- **아이디어 보드 `1:1` 버튼 제거** — 화면 맞춤 기능과 역할이 겹쳐 줌 도구에서 `1:1` 버튼을 뺐습니다.
- **메모지 컬러 선택 화면 개선**
  - 미리보기 문구가 `\n` 글자 그대로 보이던 문제를 고쳐 줄바꿈이 제대로 표시됩니다.
  - 색 스와치가 작은 알약 모양으로 카드 안에 떠 보이던 것을 **셀을 꽉 채운 둥근 컬러 타일**로 바꿔 한눈에 들어오게 했습니다.
- **이미지 배경 · 이미지 업로드 배경 화면 개선**
  - 오버레이 컬러 스와치를 위와 동일한 컬러 타일 방식으로 정리했습니다.
  - 제공 이미지·업로드 이미지 썸네일 위에 얹히던 `▧` 아이콘과 격자 무늬를 없애 그림이 깨끗하게 보입니다. (이미지가 없는 자리에는 안내 아이콘을 그대로 유지)

## 참고

- 모든 색 선택 · 배경 적용 기능은 그대로이며 표시(디자인)만 다듬은 업데이트입니다.
