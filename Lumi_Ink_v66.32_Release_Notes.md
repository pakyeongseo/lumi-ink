# 개발 빌드 v66.32 · 정규식 입력 비율·자유 메모 삽입 도구 통일

> 사용자 배포 표기는 **v1.5**를 유지합니다.
> 개발 빌드: **v66.32**

## 정규식 작업실

- `IN · findRegex` 입력 영역을 기본 높이의 약 절반인 단일 정규식용 얕은 코드 스트립으로 조정했습니다. 가로 스크롤과 수동 세로 확장은 그대로 지원합니다.
- `OUT · replaceString` 입력 영역은 기존보다 조금 높여 마크업 작성 여유를 확보했습니다.
- `Sample` 입력 영역은 기존보다 약 1.8배 높여 실제 매치 검증 문단을 보기 좋게 작성할 수 있습니다.
- 내보내기 옵션의 제목 칩은 기존 대비 약 60% 크기의 더 작은 보조 정보로 조정했습니다.

## 자유 메모

- 툴바 `＋ → 구분선`은 즉시 선을 넣는 대신, 아이디어 보드와 동일한 순서로 **굵기 → 디자인 → 컬러**를 선택하는 팝업을 엽니다.
  - 디자인 32종, 굵기 1~12px, 테마 팔레트·직접 색상을 지원합니다.
  - 저장된 자유 메모 본문에는 선택한 디자인·굵기·색상이 함께 보관됩니다.
- `＋ → 내 글 바로가기` 선택기는 아이디어 보드의 메모 링크 선택기와 동일한 UI로 통일했습니다.
  - 전체보기/프로젝트별 탭, 프로젝트 썸네일, 제목·내용·종류 검색, 프로젝트 내부 뒤로가기를 동일하게 지원합니다.
