# Lumi Ink v1.5 · 개발 빌드 v66.9

## JSON 파일 열기

- 사이드바 **열기**에서 `.json` 파일을 선택하면, 더는 자동으로 월드인포만 시도하지 않습니다.
- 일반 JSON은 다음 두 방식 중 선택할 수 있습니다.
  - **메모로 내용 보기**: 들여쓰기 정리된 JSON을 안전한 코드 블록으로 생성합니다.
  - **HTML 작업실로 원본 열기**: 공백과 정렬을 포함한 원문 문자열을 그대로 보관·편집합니다.
- World Info JSON은 기존 **로어북으로 가져오기**를 유지하며, 동시에 메모/원본 작업실로 확인할 수도 있습니다.
- 루미잉크 프로젝트 JSON은 기존 새 프로젝트 가져오기와 함께 내용 확인 경로도 제공합니다.
- JSON 문법이 깨진 파일도 구조형 가져오기는 막되, 원문을 메모 또는 HTML 작업실로 열어 확인할 수 있습니다.
- 파일 크기는 앱의 안전한 원본 편집 한도와 동일하게 **5MB 이하**로 제한합니다.

## 호환성

- 기존 HTML 불러오기와 로어북(World Info) 가져오기 방식은 유지됩니다.
- 새로 생성하는 JSON 메모/작업실 제목은 파일명에서 `.json` 확장자를 제외해 고정 저장합니다.
