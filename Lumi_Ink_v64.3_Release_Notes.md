# Lumi Ink v64.3 — 새 메모 팝업 폴리시

기준 빌드: `lumi-ink-v64_2-new-memo-hub-backup-retention`

## 수정 사항

### 새 메모 팝업

- 상단 빠른 생성 카드의 제목을 **자유 메모**로 정리했습니다.
  - 이전의 “그라데이션” 문구는 제목과 접근성 레이블에서 제거했습니다.
- 빠른 생성 카드 아래로 길게 퍼지던 그림자를 축소하고, 불투명도를 낮춰 더 가볍게 보이도록 조정했습니다.
- 캐릭터 탭의 **페르소나** 설명을 다음으로 수정했습니다.
  - `단일 페르소나 카드 · 국문/영문 · 이미지`
- 모든 탭의 생성 항목을 폭 100%의 동일한 카드 레일에 배치했습니다.
  - 짧은 제목/설명이어도 카드 폭이 흔들리지 않습니다.
  - 카드 내부 텍스트 영역도 유동 폭으로 고정해 오른쪽 화살표와 간격을 안정화했습니다.
- 생성 카드 제목·아이콘은 선택된 테마의 색을 어둡게 눌러 만든 전용 잉크색으로 적용했습니다.
  - 다크 테마에서도 대비가 유지되도록 카드 면은 밝은 테마 종이색으로 정리했습니다.
