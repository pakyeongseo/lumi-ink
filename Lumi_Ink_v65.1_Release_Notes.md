# Lumi Ink v65.1 — 라이트 모드 직접 지정 색상 보존 조정

사용자 배포 버전은 **v1.5**이며, 이 문서는 개발 추적용 **v65.1** 변경 기록입니다.

## 직접 지정 색상 구조 조정

- 기존 v65의 두 색 자동 변환 방식과 전역 테마 토큰 덮어쓰기를 제거했습니다.
- 사용자 지정은 이제 **밝게 모드 전용**입니다. 다크 모드는 현재 선택한 프리셋을 온전히 유지합니다.
- 로고, 기본 아이콘, 프로젝트·메모 기본 썸네일, 버튼·퀵 메뉴의 기존 그라데이션, 프리셋별 장식 색상은 직접 지정 기능이 더 이상 건드리지 않습니다.
- 직접 지정 화면에서 다음 구조 색상을 각각 입력한 HEX 그대로 설정할 수 있습니다.
  - 앱·보조·문서 바탕
  - 카드·보조 카드·눌림 표면
  - 상단바·상단바 보조·상단바 경계
  - 본문·보조·희미한 글자
  - 기본·옅은 경계선
- 자동 색상 보정과 보조색 자동 생성 기능을 제거했습니다. 설정값은 변형되지 않고 그대로 적용됩니다.
- **현재 프리셋 라이트 색상 불러오기** 버튼으로, 현재 프리셋의 원래 밝은 색상을 항목별 기본값으로 다시 채울 수 있습니다.

## 호환·저장

- v65의 기존 직접 지정 데이터는 가장 가까운 프리셋을 기준으로 자동 이전합니다. 이전의 두 색 값은 보존하되, 전역 색상 덮어쓰기는 중단하여 기존 디자인을 복원합니다.
- 직접 지정 사용 여부와 각 색상 역할은 IndexedDB, 로컬 설정, 전체 백업, 자동 백업에 함께 저장됩니다.
- 서비스 워커 캐시 키를 `ink-memo-v1.5-v65.1-light-custom-preserve`로 갱신했습니다.
