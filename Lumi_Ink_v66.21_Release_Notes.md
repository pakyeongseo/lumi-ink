# 개발 빌드 v66.21 · 코드 작업실 Markdown 렌더링 미리보기

- 코드 작업실에서 Markdown 원문을 열면 **PREVIEW**와 **SPLIT** 보기에서 제목, 목록, 인용문, 체크리스트, 표, 코드 블록, 링크, 이미지를 렌더링한 문서로 확인할 수 있습니다.
- Markdown 원문은 SOURCE에서 줄바꿈·공백을 포함해 그대로 편집·보관합니다.
- Markdown 안의 HTML 태그와 스크립트는 실행하지 않고 텍스트로 처리하여, HTML 샌드박스 미리보기와 별도의 안전 경계를 유지합니다.
- 파일 저장에서 Markdown을 고르면 실제로 `.md` 확장자와 `text/markdown;charset=utf-8` MIME으로 저장하도록 분기를 보정했습니다.
- HTML의 기존 샌드박스 미리보기와 JSON의 원문 보기는 변경하지 않았습니다.
