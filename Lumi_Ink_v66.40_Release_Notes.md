# 개발 빌드 v66.40 · 정규식 샘플 표시 및 퀵메뉴 슬롯 설정

- 정규식 작업실 sample 입력칸의 `[[값]]` 시각화가 후속 텍스트 폭과 입력 위치를 밀지 않도록 수정했습니다.
- 완성된 `[[값]]` 표식은 시각적으로 `값`만 보이도록 처리했습니다. 실제 sample 원문에는 `[[ ]]`가 그대로 유지됩니다.
- `샘플→IN`은 sample만 채워져 있으면 동작하도록 경로를 명시적으로 보강했습니다.
- 정규식 작업실 옵션 소제목 디자인을 상단 `REGEX` 배지와 같은 형태로 맞췄습니다.
- 퀵메뉴 설정에 슬롯 개수 조정 옵션을 추가했습니다. 1~10개까지 설정할 수 있고 기본값은 5개입니다.
- 서비스 워커 캐시명을 갱신해 모바일/PWA 환경에서 이번 수정본이 바로 반영되도록 했습니다.
