# Lumi Ink v66.3 — 퀵메뉴 투명 아이콘 · 테마 롤 확장 · 로그 세트 정리

- 퀵메뉴에서 내장 SVG 및 SVG/CSS 코드 아이콘을 선택하면 슬롯의 자동 배경, 테두리, 그림자를 제거합니다. 아이콘 자체만 투명한 영역 위에 표시됩니다.
- 기능 바로가기에 **앱 새로고침**을 추가했습니다. 먼저 보류 중인 저장을 마친 뒤 현재 앱 화면을 다시 불러옵니다.
- 사용자 지정 테마에 **사이드바 하단 버튼 배경**, **설정 그룹 제목**, **설정 항목 제목** 색상 롤을 추가했습니다. 라이트/다크 각각에서 조절 가능하며, 추천 팔레트 생성과 JSON 저장·불러오기에도 포함됩니다.
- 모든 메모 상단 제목의 더블클릭/더블터치 안내점은 제거했습니다. 제목 변경 동작은 유지됩니다.
- 로그 이름 치환 세트 5개 버튼을 작은 제본 탭처럼 재디자인했습니다. 간결한 번호와 상태점으로 구성 여부를 구분합니다.

개발 기준: v66.3 / 사용자 배포 기준: v1.5
