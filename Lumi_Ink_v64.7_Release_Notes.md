# Lumi Ink v64.7 변경사항

## 페르소나 — 페르소나 모음 병합 데이터 보존 정정

단일 페르소나 여러 개를 선택해 **페르소나 모음으로 합치기** 했을 때, 이미 카드형 데이터로 전환된 페르소나의 페이지 배열을 구형 변환기가 다시 처리하며 빈 페이지로 바꾸던 문제를 수정했습니다.

- **단일 페르소나 데이터 완전 복제** — 국문·영문 이름, 태그, 상세 설명, 세로 포트레이트, 정사각 썸네일, 갤러리, 크리에이터 메모가 새 페르소나 모음의 각 페이지로 그대로 복사됩니다.
- **기존 페르소나 모음도 안전하게 처리** — 페이지 기반 데이터가 있는 페르소나는 캐릭터 모음과 동일한 복제 경로를 사용합니다.
- **구형 데이터 호환 유지** — 아직 구형 단일 페르소나 구조인 기록은 이전과 같이 카드형 데이터로 변환해 병합합니다.
- 원본 페르소나는 병합 후에도 삭제되거나 수정되지 않습니다.

## 배포 갱신

- 서비스 워커 캐시 키를 `ink-memo-v1.5-v64.7-persona-merge-data-fix`로 갱신했습니다.
